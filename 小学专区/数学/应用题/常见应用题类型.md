# 小学数学常见应用题类型

> 按题型分类，配有解题步骤

## 一、和差问题

已知两个数的和与差，求这两个数。

**公式：**
- 大数 = (和 + 差) / 2
- 小数 = (和 - 差) / 2

**例题：** 两个数的和是80，差是20，求这两个数。
- 大数 = (80 + 20) / 2 = 50
- 小数 = (80 - 20) / 2 = 30

## 二、和倍问题

已知两个数的和与倍数关系，求这两个数。

**公式：**
- 小数 = 和 / (倍数 + 1)
- 大数 = 小数 * 倍数

**例题：** 两个数的和是120，大数是小数的3倍，求这两个数。
- 小数 = 120 / (3 + 1) = 30
- 大数 = 30 * 3 = 90

## 三、差倍问题

已知两个数的差与倍数关系，求这两个数。

**公式：**
- 小数 = 差 / (倍数 - 1)
- 大数 = 小数 * 倍数

**例题：** 大数比小数多60，大数是小数的4倍，求这两个数。
- 小数 = 60 / (4 - 1) = 20
- 大数 = 20 * 4 = 80

## 四、行程问题

### 基本公式
- 路程 = 速度 x 时间
- 速度 = 路程 / 时间
- 时间 = 路程 / 速度

### 相遇问题
- 相遇路程 = (甲速度 + 乙速度) x 相遇时间

### 追及问题
- 追及路程 = (快速度 - 慢速度) x 追及时间

**例题：** 甲乙两地相距360千米，A车每小时行60千米，B车每小时行40千米，同时相向出发，几小时后相遇？
- 相遇时间 = 360 / (60 + 40) = 3.6小时

## 五、工程问题

**公式：**
- 工作量 = 工作效率 x 工作时间
- 合作效率 = 甲效率 + 乙效率

**例题：** 一项工程，甲单独做需10天，乙单独做需15天，两人合做需几天？
- 甲效率 = 1/10，乙效率 = 1/15
- 合作效率 = 1/10 + 1/15 = 5/30 = 1/6
- 合作时间 = 1 / (1/6) = 6天

## 六、浓度问题

**公式：**
- 浓度 = 溶质 / 溶液 x 100%
- 溶液 = 溶质 + 溶剂

**例题：** 100克10%的盐水，加入多少克水变成5%的盐水？
- 原来盐 = 100 x 10% = 10克
- 10 / (100 + x) = 5%
- 100 + x = 200
- x = 100克

## 七、利润问题

**公式：**
- 利润 = 售价 - 成本
- 利润率 = 利润 / 成本 x 100%
- 售价 = 成本 x (1 + 利润率)
- 打折后售价 = 原价 x 折扣

**例题：** 一件商品成本80元，加价50%后打八折出售，实际利润率是多少？
- 标价 = 80 x (1 + 50%) = 120元
- 售价 = 120 x 0.8 = 96元
- 利润 = 96 - 80 = 16元
- 利润率 = 16/80 x 100% = 20%
