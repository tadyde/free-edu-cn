# 小学5年级小数运算训练

## 小数加减法

### 计算要点
- 小数点对齐
- 位数不够用0补齐

### 练习
1. 3.45 + 2.78 = 6.23
2. 8.90 - 3.47 = 5.43
3. 12.5 + 7.86 = 20.36
4. 15.00 - 8.73 = 6.27
5. 23.456 + 7.544 = 31.000

## 小数乘法

### 计算要点
- 先按整数乘法计算
- 再数小数位数，从右往左数

### 练习
1. 0.3 * 0.4 = 0.12
2. 2.5 * 0.4 = 1.00
3. 1.25 * 0.8 = 1.00
4. 3.6 * 2.5 = 9.00
5. 0.125 * 8 = 1.00

## 小数除法

### 计算要点
- 先移动除数的小数点使它变成整数
- 除数移动几位，被除数也移动几位
- 再按整数除法计算

### 练习
1. 7.2 / 0.8 = 9
2. 3.6 / 1.2 = 3
3. 0.96 / 0.12 = 8
4. 4.5 / 0.15 = 30
5. 6.25 / 2.5 = 2.5

## 应用题

1. 小明买了3.5千克苹果，每千克4.8元，应付多少元？答：16.8元
2. 一根绳子长7.5米，剪成0.5米的小段，能剪几段？答：15段
3. 一箱饮料24瓶，共重12.6千克，每瓶重多少千克？答：0.525千克
