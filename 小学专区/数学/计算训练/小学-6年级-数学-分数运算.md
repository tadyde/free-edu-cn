# 小学6年级分数运算训练

## 分数加减法

### 同分母
1/5 + 2/5 = 3/5
3/7 - 1/7 = 2/7

### 异分母（先通分）
1/3 + 1/4 = 4/12 + 3/12 = 7/12
2/3 - 1/6 = 4/6 - 1/6 = 3/6 = 1/2

### 练习
1. 1/2 + 1/3 = 5/6
2. 3/4 - 1/3 = 5/12
3. 2/5 + 3/10 = 7/10
4. 5/6 - 1/4 = 7/12
5. 1/2 + 1/4 + 1/8 = 7/8

## 分数乘法

### 法则
分子乘分子，分母乘分母（能约分先约分）

### 练习
1. 2/3 * 3/4 = 1/2
2. 4/5 * 5/8 = 1/2
3. 3/7 * 14/9 = 2/3
4. 5/6 * 12/25 = 2/5
5. 7/8 * 4/21 = 1/6

## 分数除法

### 法则
除以一个分数等于乘以它的倒数

### 练习
1. 2/3 / 4/5 = 2/3 * 5/4 = 5/6
2. 3/4 / 9/8 = 3/4 * 8/9 = 2/3
3. 5/6 / 5/12 = 5/6 * 12/5 = 2
4. 1/2 / 1/4 = 2
5. 7/9 / 14/27 = 7/9 * 27/14 = 3/2

## 混合运算

先乘除后加减，有括号先算括号内

1. 1/2 + 1/3 * 3/4 = 1/2 + 1/4 = 3/4
2. (1/2 + 1/3) * 3/4 = 5/6 * 3/4 = 5/8
3. 2/3 - 1/4 / 1/2 = 2/3 - 1/2 = 1/6

## 应用题

1. 一桶油用去了3/5，还剩几分之几？答：2/5
2. 一条路长12千米，修了3/4，修了多少千米？答：9千米
3. 一箱苹果24个，小丽吃了1/6，吃了几个？答：4个
