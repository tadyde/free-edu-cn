# 小学4年级竖式除法训练

4年级重点掌握两位数除法。

## 除法练习

### 一位数除法
1. 84/7=12
2. 96/8=12
3. 75/5=15
4. 91/7=13
5. 108/9=12

### 两位数除法（整除）
1. 144/12=12
2. 169/13=13
3. 196/14=14
4. 225/15=15
5. 256/16=16

### 有余数除法
1. 100/7=14余2
2. 85/9=9余4
3. 137/11=12余5
4. 200/13=15余5
5. 99/8=12余3

## 应用题

1. 168本图书分给12个班，每班14本
2. 95颗糖每8颗一袋，装11袋余7颗
3. 360米路每天修15米，需24天
4. 325个苹果分给25人，每人13个

## 竖式步骤

1. 从高位除起
2. 除到哪位商哪位
3. 每次余数要比除数小
