# 小学数学公式汇总

## 一、运算定律

### 加法
- 交换律：a+b = b+a
- 结合律：(a+b)+c = a+(b+c)

### 乘法
- 交换律：a*b = b*a
- 结合律：(a*b)*c = a*(b*c)
- 分配律：a*(b+c) = a*b + a*c

## 二、几何公式

### 长方形
- 周长 = 2*(长+宽)
- 面积 = 长*宽

### 正方形
- 周长 = 4*边长
- 面积 = 边长*边长

### 三角形
- 面积 = 底*高/2

### 平行四边形
- 面积 = 底*高

### 梯形
- 面积 = (上底+下底)*高/2

### 圆
- 周长 = 2*pi*r = pi*d
- 面积 = pi*r^2

### 长方体
- 表面积 = 2*(长*宽+长*高+宽*高)
- 体积 = 长*宽*高

### 正方体
- 表面积 = 6*棱长^2
- 体积 = 棱长^3

### 圆柱
- 侧面积 = 2*pi*r*h
- 表面积 = 2*pi*r^2 + 2*pi*r*h
- 体积 = pi*r^2*h

### 圆锥
- 体积 = pi*r^2*h/3

## 三、数量关系

- 行程：路程=速度*时间
- 工程：工作量=效率*时间
- 价格：总价=单价*数量
- 利润：利润=售价-成本
- 利润率：利润率=利润/成本*100%
- 利息：利息=本金*利率*时间

## 四、单位换算

### 长度
1千米=1000米 1米=10分米=100厘米=1000毫米

### 面积
1平方千米=100公顷 1公顷=10000平方米 1平方米=100平方分米

### 体积
1立方米=1000立方分米 1立方分米=1升=1000毫升

### 重量
1吨=1000千克 1千克=1000克
