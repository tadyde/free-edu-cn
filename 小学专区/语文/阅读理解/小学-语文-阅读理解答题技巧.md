# 小学语文阅读理解答题技巧

适用于3-6年级。

## 一、常见题型

1. **提取信息题**：直接从原文找答案
2. **概括题**：谁+做了什么+结果如何
3. **词语理解题**：本义+文中义
4. **感想题**：观点+原文依据+联系生活

## 二、答题公式

- 概括段意 = 这段话写了（谁/什么）+（怎么样/做了什么）
- 人物形象 = 从XX事例可以看出XX是一个（形容词）的人
- 中心思想 = 本文通过记叙XX表达了XX
- 句子赏析 = 用了XX修辞手法写出了XX表达了XX

## 三、常见修辞手法

| 修辞 | 特征 | 作用 |
|------|------|------|
| 比喻 | 把A比作B | 生动形象 |
| 拟人 | 把物当人写 | 亲切有趣 |
| 排比 | 三个以上相同句式 | 增强气势 |
| 夸张 | 放大或缩小 | 突出特征 |
| 反问 | 用疑问表肯定/否定 | 加强语气 |
| 设问 | 自问自答 | 引起注意 |

## 四、考试提醒

- 先读题再读文章
- 答案尽量用原文词句
- 分点作答条理清晰
- 宁可多写不可不写
