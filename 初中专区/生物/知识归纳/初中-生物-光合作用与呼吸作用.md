# 初中生物：光合作用与呼吸作用

## 光合作用

### 反应式
6CO2 + 6H2O →(光能/叶绿体) C6H12O6 + 6O2

### 条件
- 场所：叶绿体
- 条件：光照
- 原料：CO2 + H2O
- 产物：有机物 + O2

### 意义
1. 制造有机物，提供食物来源
2. 释放氧气，维持碳氧平衡
3. 将光能转变为化学能储存

## 呼吸作用

### 反应式
C6H12O6 + 6O2 → 6CO2 + 6H2O + 能量

### 条件
- 场所：所有活细胞的线粒体
- 条件：时刻进行（有光无光都可）
- 原料：有机物 + O2
- 产物：CO2 + H2O + 能量

### 意义
为生命活动提供能量

## 对比

| 项目 | 光合作用 | 呼吸作用 |
|------|---------|---------|
| 场所 | 叶绿体 | 线粒体 |
| 条件 | 需光 | 不需光 |
| 原料 | CO2+H2O | 有机物+O2 |
| 产物 | 有机物+O2 | CO2+H2O+能量 |
| 物质变化 | 无机物→有机物 | 有机物→无机物 |
| 能量变化 | 光能→化学能 | 化学能→生命活动能量 |

## 相互关系

光合作用和呼吸作用相互依存、相互对立。光合作用积累有机物，呼吸作用消耗有机物。白天光合作用大于呼吸作用，植物生长；夜晚只有呼吸作用。
