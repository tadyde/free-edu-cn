# 初中英语被动语态

## 一、基本结构

be + 过去分词(done)

## 二、各时态的被动语态

| 时态 | 主动 | 被动 |
|------|------|------|
| 一般现在时 | do/does | am/is/are done |
| 一般过去时 | did | was/were done |
| 一般将来时 | will do | will be done |
| 现在完成时 | have done | have been done |
| 情态动词 | can do | can be done |

## 三、例句

### 一般现在时
Active: They clean the room every day.
Passive: The room is cleaned every day.

### 一般过去时
Active: He wrote the letter yesterday.
Passive: The letter was written yesterday.

### 一般将来时
Active: We will plant trees tomorrow.
Passive: Trees will be planted tomorrow.

### 情态动词
Active: You must finish the work.
Passive: The work must be finished.

## 四、特殊情况

1. 不及物动词没有被动语态
2. 短语动词变被动时不能丢掉介词/副词
   - look after → be looked after
3. 双宾语动词可有两种被动形式
   - give sb sth → sb be given sth / sth be given to sb
4. 使役动词/感官动词
   - make sb do → sb be made to do（要加to）
   - see sb do → sb be seen to do

## 五、主动变被动步骤

1. 宾语变主语
2. 谓语变be+done（注意时态）
3. 主语变by+宾格（可省略）
