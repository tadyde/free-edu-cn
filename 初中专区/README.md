# 初中专区

面向初中7-9年级学生的学习资料。

## 学科目录

- [语文](语文/) - 阅读理解、文言文、写作技巧
- [数学](数学/) - 代数、几何、压轴题解析
- [英语](英语/) - 语法体系、阅读训练、写作模板
- [物理](物理/) - 力学、电学、实验专题
- [化学](化学/) - 元素周期、化学方程、实验要点

## 年级划分

| 年级 | 语文 | 数学 | 英语 | 物理 | 化学 |

| 九年级 | 中考复习 | 综合压轴 | 中考冲刺 | 中考实验 | 中考冲刺 |
