# 初中物理公式大全

## 力学

### 速度
v = s/t

### 密度
rho = m/V

### 重力
G = mg（g=9.8N/kg，近似取10）

### 压强
p = F/S

### 液体压强
p = rho*g*h

### 浮力
F浮 = rho液*g*V排（阿基米德原理）
F浮 = G - F拉（称重法）
漂浮/悬浮：F浮 = G

### 功
W = F*s

### 功率
P = W/t

### 机械效率
eta = W有用/W总 * 100%

### 杠杆平衡
F1*L1 = F2*L2

## 热学

### 比热容
Q = c*m*(t-t0)

### 热值
Q = m*q（完全燃烧）

### 热机效率
eta = W有用/Q释放 * 100%

## 电学

### 欧姆定律
I = U/R

### 串联
I=I1=I2  U=U1+U2  R=R1+R2

### 并联
I=I1+I2  U=U1=U2  1/R=1/R1+1/R2

### 电功
W = UIt = I^2Rt = U^2t/R = Pt

### 电功率
P = UI = I^2R = U^2/R

### 焦耳定律
Q = I^2Rt
