# 初中9年级物理电学基础

## 一、电荷

- 摩擦起电：不同物体摩擦后带电
- 两种电荷：正电荷（+）、负电荷（-）
- 同种电荷相互排斥，异种电荷相互吸引
- 验电器：检测物体是否带电

## 二、电路

### 基本组成
电源 + 导线 + 开关 + 用电器

### 电路状态
- 通路：正常工作
- 断路：某处断开，无电流
- 短路：不经过用电器直接连通，危险！

### 串联与并联

| 特征 | 串联 | 并联 |
|------|------|------|
| 电流 | I=I1=I2 | I=I1+I2 |
| 电压 | U=U1+U2 | U=U1=U2 |
| 电阻 | R=R1+R2 | 1/R=1/R1+1/R2 |
| 开关 | 一个控制所有 | 各支路独立控制 |

## 三、欧姆定律

I = U / R

- I：电流（安培A）
- U：电压（伏特V）
- R：电阻（欧姆ohm）

## 四、电功率

P = UI = I^2*R = U^2/R

- P：功率（瓦特W）
- 1度电 = 1kWh = 3.6*10^6 J

## 五、安全用电

- 家庭电路电压：220V
- 安全电压：不高于36V
- 保险丝：电流过大时熔断，保护电路
- 三孔插座：左零右火上接地
