# 初中文言文高频词汇

## 一、实词

### 常见一词多义

| 词 | 常见义 | 例句 |
|----|--------|------|
| 之 | 往/到/的/他/它/无义 | 予独爱莲之出淤泥而不染 |
| 其 | 他的/那/其中/难道 | 其真无马邪 |
| 以 | 用/凭/把/因为/来 | 以刀劈狼首 |
| 于 | 在/到/向/对/比 | 苛政猛于虎 |
| 而 | 并且/却/就/地 | 敏而好学 |
| 为 | 做/成为/被/替/给 | 为人所讥 |

## 二、虚词

### 之
1. 作代词：他/它（肉食者谋之）
2. 作助词-的：莲之出淤泥（结构）/ 予独爱莲之出（取消句子独立性）/ 怅恨久之（音节）
3. 作动词：往（吾欲之南海）

### 其
1. 作代词：他的/它的/那（其岸势犬牙差互）
2. 作副词：难道/大概（其如土石何）
3. 作连词：如果/假设

### 以
1. 介词：用/拿/凭/把/因（以光先帝遗德）
2. 连词：来/用来/因为

### 而
1. 并列：又/且（黑质而白章）
2. 承接：就/然后（温故而知新）
3. 转折：却/但是（学而不思则罔）
4. 修饰：地（呼尔而与之）
