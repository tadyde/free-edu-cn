# 初中9年级化学实验基本操作

## 一、常见仪器

| 仪器 | 用途 |
|------|------|
| 试管 | 少量试剂反应容器 |
| 烧杯 | 较大量试剂反应/配制溶液 |
| 量筒 | 量取液体体积 |
| 酒精灯 | 加热 |
| 漏斗 | 过滤 |
| 铁架台 | 固定仪器 |
| 蒸发皿 | 蒸发浓缩 |
| 集气瓶 | 收集气体 |

## 二、基本操作

### 药品取用
- 固体：药匙或纸槽，试管横放，慢慢竖起
- 液体：瓶塞倒放，标签朝手心，瓶口紧挨试管口

### 加热
- 试管加热：先预热，后集中加热
- 液体不超过试管容积的1/3
- 试管口不对着人

### 过滤
- 一贴：滤纸紧贴漏斗壁
- 二低：滤纸低于漏斗边缘，液面低于滤纸
- 三靠：烧杯嘴靠玻璃棒，玻璃棒靠滤纸，漏斗下端靠烧杯壁

### 蒸发
- 蒸发皿放在铁架台的铁圈上
- 用玻璃棒不断搅拌
- 出现较多固体时停止加热

## 三、气体制备

### 氧气（加热高锰酸钾）
2KMnO4 →(加热) K2MnO4 + MnO2 + O2

### 二氧化碳（石灰石与稀盐酸）
CaCO3 + 2HCl → CaCl2 + H2O + CO2

### 收集方法
- 排水法：不易溶于水的气体（O2）
- 向上排空气法：密度比空气大（CO2）
- 向下排空气法：密度比空气小（H2）
