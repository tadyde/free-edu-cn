# 初中化学常见化学方程式

> 依据人教版初中化学教材整理，按章节分类

## 氧气的制取

1. 加热高锰酸钾制氧气
   2KMnO4 --加热--> K2MnO4 + MnO2 + O2

2. 过氧化氢制氧气（二氧化锰催化）
   2H2O2 --MnO2--> 2H2O + O2

3. 加热氯酸钾制氧气（二氧化锰催化）
   2KClO3 --MnO2,加热--> 2KCl + 3O2

## 碳和碳的氧化物

4. 碳在氧气中充分燃烧
   C + O2 --点燃--> CO2

5. 碳在氧气中不充分燃烧
   2C + O2 --点燃--> 2CO

6. 二氧化碳与水反应
   CO2 + H2O === H2CO3

7. 二氧化碳与澄清石灰水反应
   CO2 + Ca(OH)2 === CaCO3 + H2O

8. 一氧化碳还原氧化铜
   CO + CuO --加热--> Cu + CO2

9. 碳酸分解
   H2CO3 === H2O + CO2

## 金属

10. 铁在氧气中燃烧
    3Fe + 2O2 --点燃--> Fe3O4

11. 铁与稀盐酸反应
    Fe + 2HCl === FeCl2 + H2

12. 铁与稀硫酸反应
    Fe + H2SO4 === FeSO4 + H2

13. 铁与硫酸铜溶液反应（湿法炼铜）
    Fe + CuSO4 === FeSO4 + Cu

14. 铝与稀盐酸反应
    2Al + 6HCl === 2AlCl3 + 3H2

15. 镁在空气中燃烧
    2Mg + O2 --点燃--> 2MgO

## 酸的性质

16. 盐酸与氢氧化钠反应
    HCl + NaOH === NaCl + H2O

17. 硫酸与氢氧化钠反应
    H2SO4 + 2NaOH === Na2SO4 + 2H2O

18. 盐酸与碳酸钠反应
    2HCl + Na2CO3 === 2NaCl + H2O + CO2

19. 盐酸与碳酸钙反应
    2HCl + CaCO3 === CaCl2 + H2O + CO2

## 碱的性质

20. 氢氧化钠与二氧化碳反应
    2NaOH + CO2 === Na2CO3 + H2O

21. 氢氧化钙与二氧化碳反应
    Ca(OH)2 + CO2 === CaCO3 + H2O

22. 氢氧化钠与硫酸铜反应
    2NaOH + CuSO4 === Cu(OH)2 + Na2SO4

## 盐的性质

23. 氯化钠与硝酸银反应
    NaCl + AgNO3 === AgCl + NaNO3

24. 硫酸钠与氯化钡反应
    Na2SO4 + BaCl2 === BaSO4 + 2NaCl

## 燃烧与灭火

25. 甲烷燃烧
    CH4 + 2O2 --点燃--> CO2 + 2H2O

26. 酒精燃烧
    C2H5OH + 3O2 --点燃--> 2CO2 + 3H2O

## 记忆技巧

1. **先理解后记忆**：每个方程式都要理解反应原理
2. **注意条件**：加热、点燃、催化剂等条件不能漏
3. **配平检查**：反应前后原子种类和数目必须相等
4. **分类记忆**：按酸碱盐、金属、气体等分类记忆效果更好
5. **注意沉淀和气体符号**：沉淀标""，气体标""

> 注：=== 表示常温下进行，--xxx--> 表示需要特定条件
