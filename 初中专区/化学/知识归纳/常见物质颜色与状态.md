# 初中化学常见物质颜色与状态

> 考试常考，必须记住

## 固体颜色

### 白色固体
- MgO（氧化镁）
- CaO（氧化钙/生石灰）
- CaCO3（碳酸钙/石灰石）
- NaOH（氢氧化钠/烧碱）
- Na2CO3（碳酸钠/纯碱）
- NaCl（氯化钠/食盐）
- 无水CuSO4（无水硫酸铜/白色粉末）

### 黑色固体
- C（碳/炭黑）
- CuO（氧化铜）
- Fe3O4（四氧化三铁）
- MnO2（二氧化锰）

### 红色固体
- Cu（铜/紫红色）
- Fe2O3（氧化铁/红棕色）
- P（红磷/暗红色）

### 黄色固体
- S（硫/淡黄色）

### 银白色固体
- Fe（铁）
- Mg（镁）
- Al（铝）
- Zn（锌）

## 液体颜色

### 无色液体
- H2O（水）
- 稀盐酸
- 稀硫酸
- NaOH溶液
- NaCl溶液

### 蓝色溶液
- CuSO4溶液（硫酸铜溶液）
- CuCl2稀溶液

### 黄色溶液
- FeCl3溶液（氯化铁溶液）

### 浅绿色溶液
- FeSO4溶液（硫酸亚铁溶液）
- FeCl2溶液（氯化亚铁溶液）

## 沉淀颜色

### 白色沉淀
- BaSO4（硫酸钡）— 不溶于稀硝酸
- AgCl（氯化银）— 不溶于稀硝酸
- CaCO3（碳酸钙）— 溶于稀硝酸，产生气泡
- BaCO3（碳酸钡）— 溶于稀硝酸，产生气泡
- Mg(OH)2（氢氧化镁）— 溶于稀硝酸，无气泡
- Ag2CO3（碳酸银）— 溶于稀硝酸，产生气泡

### 蓝色沉淀
- Cu(OH)2（氢氧化铜）— 溶于稀硝酸，溶液变蓝

### 红褐色沉淀
- Fe(OH)3（氢氧化铁）— 溶于稀硝酸，溶液变黄

## 气体颜色

### 无色气体
- O2（氧气）
- H2（氢气）
- N2（氮气）
- CO（一氧化碳）
- CO2（二氧化碳）
- CH4（甲烷）

> 注：初中化学涉及的有色气体极少，常见气体基本均为无色。

## 特殊现象

| 现象 | 涉及物质 |
|------|----------|
| 白色粉末变蓝 | 无水CuSO4遇水变蓝（CuSO4·5H2O） |
| 铁在氧气中燃烧"火星四射" | Fe → Fe3O4，生成黑色固体 |
| 镁在空气中燃烧"耀眼白光" | Mg → MgO，生成白色固体 |
| 铜丝插入硝酸银溶液 | Cu + AgNO3 → 表面附着银白色物质，溶液变蓝 |
| 铁钉插入硫酸铜溶液 | Fe + CuSO4 → 表面附着红色物质，溶液变浅绿色 |

---

> 颜色和状态是鉴别物质的重要依据，建议结合实验现象一起记忆。
