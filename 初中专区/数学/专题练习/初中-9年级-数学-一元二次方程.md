# 初中9年级一元二次方程专题

## 一、标准形式

ax^2 + bx + c = 0（a不等于0）

## 二、解法

### 1. 因式分解法
x^2 - 5x + 6 = 0
(x-2)(x-3) = 0
x=2 或 x=3

### 2. 配方法
x^2 + 6x + 5 = 0
x^2 + 6x = -5
x^2 + 6x + 9 = 4
(x+3)^2 = 4
x+3 = 2 或 x+3 = -2
x = -1 或 x = -5

### 3. 公式法
x = (-b +/- sqrt(b^2-4ac)) / (2a)

判别式：delta = b^2 - 4ac
- delta>0：两个不等实根
- delta=0：两个相等实根
- delta<0：无实根

## 三、韦达定理

x1 + x2 = -b/a
x1 * x2 = c/a

## 四、练习题

1. x^2-7x+12=0 （答案：x=3或x=4）
2. 2x^2-3x-2=0 （答案：x=2或x=-1/2）
3. x^2+4x+4=0 （答案：x=-2，重根）
4. 3x^2-5x+1=0 （答案：x=(5+/-sqrt(13))/6）
5. x^2-2x-8=0 （答案：x=4或x=-2）
