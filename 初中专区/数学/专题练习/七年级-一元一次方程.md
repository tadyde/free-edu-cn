# 七年级数学专题：一元一次方程

> 难度：基础→提高 | 知识点：一元一次方程的解法与应用 | 建议时间：40分钟

## 一、解方程

### 基础题
1. 3x + 5 = 17
2. 2x - 8 = 6
3. 4x + 3 = 2x + 11
4. 5x - 2 = 3x + 8
5. 7 - 2x = 3x + 2

### 提高题
6. 3(x - 2) = 2(x + 1)
7. 5 - 2(3x - 1) = 3x + 2
8. (2x + 1)/3 - (x - 1)/2 = 1
9. (3x - 1)/4 - 1 = (2x + 1)/3
10. 2 - [3x - 2(x - 1)] = 5

## 二、应用题

### 行程问题
1. 甲乙两地相距240千米，一辆汽车从甲地出发，每小时行60千米，另一辆汽车从乙地出发，每小时行40千米，两车相向而行，几小时后相遇？

### 工程问题
2. 一项工程，甲单独做需12天完成，乙单独做需18天完成。两人合作几天可以完成？

### 利润问题
3. 某商品进价为200元，标价为360元，打折售后仍获利20%。该商品是打几折出售的？

### 分配问题
4. 将一些笔记本分给若干名学生，每人3本还剩12本，每人4本还差6本。共有多少名学生？多少本笔记本？

### 年龄问题
5. 父亲今年45岁，儿子今年15岁，几年前父亲的年龄是儿子年龄的5倍？

---

## 解题步骤提示

解一元一次方程的一般步骤：
1. **去分母**：方程两边同乘最小公倍数
2. **去括号**：利用分配律展开
3. **移项**：把含未知数的项移到左边，常数项移到右边（移项要变号）
4. **合并同类项**
5. **系数化为1**：两边同除以未知数的系数

## 参考答案

### 解方程
1. x = 4
2. x = 7
3. x = 4
4. x = 5
5. x = 1
6. x = 8
7. x = 1/3
8. x = -1
9. x = 1/5
10. x = -1

### 应用题
1. 设x小时后相遇。60x + 40x = 240，x = 2.4小时
2. 设两人合作x天完成。(1/12 + 1/18)x = 1，x = 36/5 = 7.2天
3. 设打x折。360 x (x/10) = 200 x (1 + 20%)，36x = 240，x = 20/3 约等于6.67折
4. 设学生x人。3x + 12 = 4x - 6，x = 18名学生，笔记本 3 x 18 + 12 = 66本
5. 设x年前。(45 - x) = 5(15 - x)，45 - x = 75 - 5x，4x = 30，x = 7.5年
