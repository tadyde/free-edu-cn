# 初中7年级有理数专题

## 一、有理数分类

正整数、零、负整数 → 整数
正分数、负分数 → 分数
整数和分数统称有理数

## 二、数轴

- 三要素：原点、正方向、单位长度
- 右边的数总比左边的数大

## 三、绝对值

|a| = a（a>0）
|a| = 0（a=0）
|a| = -a（a<0）

性质：|a| >= 0，|a| = |-a|

## 四、有理数运算

### 加法法则
1. 同号相加：取相同符号，绝对值相加
2. 异号相加：取绝对值大的符号，绝对值相减
3. 互为相反数相加得0
4. 一个数加0仍得这个数

### 减法法则
减去一个数等于加上它的相反数
a - b = a + (-b)

### 乘法法则
1. 两数相乘，同号得正，异号得负
2. 任何数乘0都得0

### 除法法则
除以一个不等于0的数等于乘以它的倒数
a / b = a * (1/b)

### 乘方
(-1)^奇数 = -1
(-1)^偶数 = 1

## 五、科学记数法

a * 10^n（1<=|a|<10，n为整数）

例：345000 = 3.45 * 10^5
0.00345 = 3.45 * 10^(-3)
