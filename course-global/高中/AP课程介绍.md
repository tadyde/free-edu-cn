# AP课程全面介绍

> Advanced Placement，由美国大学理事会（College Board）主办的国际预科课程。

## 📖 AP体系概述

### 基本信息
- **适读年级**：10-12年级（美国高中）
- **课程数量**：38门（横跨6大类学科）
- **考试时间**：每年5月（全球统一）
- **评分**：1-5分
  - 5分：Extremely well qualified
  - 4分：Well qualified
  - 3分：Qualified（大学学分要求通常3+）
  - 2分：Possibly qualified
  - 1分：No recommendation

### 六大类学科

| 类别 | AP科目示例 |
|------|-----------|
| 艺术 | AP艺术与设计、AP音乐理论 |
| 英语 | AP英语语言与写作、AP英语文学与写作 |
| 历史与社会科学 | AP美国历史、AP世界历史、AP心理学、AP经济学 |
| 数学与计算机 | AP微积分AB/BC、AP统计学、AP计算机A/原理 |
| 科学 | AP物理1/2/C、AP化学、AP生物、AP环境科学 |
| 世界语言与文化 | AP中文、AP法语、AP西班牙语等 |

## 📊 最受欢迎的AP科目（按全球考生量）

1. **AP英语语言与写作** — 英语非母语者慎选
2. **AP微积分AB/BC** — 数学基础必备
3. **AP美国历史** — 文科入门之选
4. **AP心理学** — 趣味性强，词汇量大
5. **AP物理1/2/C** — 理工科必选
6. **AP计算机科学A** — CS入门友好
7. **AP经济学（宏/微观）** — 文科生友好，可同时考两门

## 💡 中国学生AP选科建议

| 学生类型 | 推荐AP科目 | 说明 |
|---------|-----------|------|
| 理工科 | 微积分BC+物理C+计算机A | 数学+物理+编程，三件套 |
| 商科 | 经济学（宏+微）+微积分BC | 经济+数学基础 |
| 生化 | 化学+生物+微积分BC | 医学/生物方向 |
| 人文 | 英语语言+历史+心理学 | 深度积累 |
| 计算机 | 计算机A+计算机原理+微积分BC | CS专业敲门砖 |

## 📌 AP vs IB vs A-Level 对比

| 维度 | AP | IBDP | A-Level |
|------|-----|------|---------|
| 课程深度 | 单科深入 | 六科均衡 | 3-4科深度 |
| 评估方式 | 全球统考（5月） | 内部+外部综合 | 全球统考 |
| 学分抵换 | 直接抵换（各校政策不同） | 部分大学可抵 | 部分可抵 |
| 英语要求 | 中等 | 高（IA/EE全英文写作）| 中等 |
| 灵活性 | 高（选任何科目组合）| 中（6组必修）| 高（可选任何科目）|
