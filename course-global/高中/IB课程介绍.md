# IB DP 课程全面介绍

> Diploma Programme，国际文凭组织（IBO）为16-19岁学生设计的两年制高中课程。

## 📖 IBDP体系概述

### 课程结构
学生需从6个学科组中选择各1门，其中：
- 至少3门为HL（Higher Level，高难度）
- 最多3门为SL（Standard Level，标准难度）

| 学科组 | HL/SL |
|--------|-------|
| 1. 语言与文学（母语） | 必选 |
| 2. 语言习得（外语） | 必选 |
| 3. 个人与社会 | 必选 |
| 4. 科学 | 必选 |
| 5. 数学 | 必选 |
| 6. 艺术（可从1-4组替代）| 选考 |

### 核心要素（Core）
| 核心 | 内容 |
|------|------|
| TOK（知识理论） | 跨学科哲学探究，100小时内，提交论文+口头报告 |
| EE（拓展论文） | 4000字独立研究论文 |
| CAS（创造/行动/服务）| 150小时课外活动（ creativity/action/service） |

### 评分体系
- 每门1-7分（HL/SL分开划线）
- 核心（TOK+EE）最高3分
- **满分45分**（38分为顶尖大学门槛，24分为及格）
- 40+分：常春藤、牛剑级别

## 📊 六大组别选科详解

### Group 1：语言与文学
- **中文A文学/语言与文学**（中国学生首选）
- HL/SL差异：阅读量、分析深度、IA要求

### Group 2：语言习得
- **英语B HL/SL**（国际学校学生普遍选择）
- 中文B（国内IB学生较少选）

### Group 3：个人与社会
- 历史、地理、经济、商务管理、哲学、信息科技
- **经济HL**：数学基础要求低，文科生友好
- **历史HL**：阅读量大，写作要求极高

### Group 4：科学
- 物理、化学、生物、环境系统
- **物理HL**：数学要求高（需微积分基础）
- **化学HL**：记忆量大，IA实验复杂

### Group 5：数学
- **数学AA（分析与方法）**：传统数学，证明导向（HL难）
- **数学AI（应用与解释）**：统计、建模导向（HL相对友好）
- 近年趋势：理工科建议选AA，文科/商科选AI

### Group 6：艺术
- 视觉艺术、音乐、戏剧、电影

## 💡 IBDP vs AP vs A-Level

| 维度 | IBDP | A-Level | AP |
|------|------|---------|-----|
| 课程广度 | 六科全覆盖 | 3-4科 | 1-多门自选 |
| 英语要求 | 最高（TOK+EE+IA全英文）| 中等 | 低（AP exam全英文）|
| 适合类型 | 全面均衡型 | 专长深入型 | 自学能力强型 |
| 大学申请 | 全球认可，含金量高 | 英联邦为主 | 美国为主，全球认可 |

## 📌 IBDP选课策略

### 理工科申请方向（MIT/斯坦福/剑桥等）
- 数学AA HL + 物理 HL + 化学/生物 HL + 英语B SL + 一门人文
- 辅助：计算机科学 HL

### 商科申请方向
- 数学 AA/AI HL + 经济 HL + 英语B + 商管 HL
- 辅助：心理学/环境科学 SL

### 文科/法学方向
- 中文A HL + 英语A HL + 历史/哲学 HL
- 辅助：数学AI SL
