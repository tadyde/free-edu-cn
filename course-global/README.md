### 🇺🇸 美国教育体系
- Common Core State Standards（CCSS）核心标准
- AP课程大纲（数学/物理/化学/计算机）
- Khan Academy等免费平台资源汇总

### 🇬🇧 英国教育体系
- National Curriculum（NC）体系简介
- GCSE / A-Level 课程结构
- 牛津/剑桥公开资源

### 🇯🇵 日本教育体系
- 小学·初中·高中课程标准概要
- 中日数学/语文教学内容对比
- 日本教育资源公开平台

- 保持内容客观中立，不做商业推广
