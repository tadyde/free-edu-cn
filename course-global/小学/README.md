# 🌍 小学国际课程

> 收录**美国、英国、日本**等主流国家的小学教育标准与优质资源。

## 📚 国家课程标准

| 国家 | 课程体系 | 核心特点 |
|------|----------|----------|
| 🇺🇸 美国 | CCSS（Common Core State Standards） | 州通用标准，强调批判性思维 |
| 🇬🇧 英国 | NC（National Curriculum） | 分关键阶段（KS1/KS2），重视读写算 |
| 🇯🇵 日本 | 学習指導要領 | 注重基础扎实，学科整合性强 |

## 📋 各学段内容

### 美国 CCSS 英语标准（K-5年级）
- **阅读**：文学文本 + 信息文本
- **写作**：叙述/信息/议论三种类型
- **语言**：词汇积累与运用
- **口语与听力**：沟通与表达

### 美国 CCSS 数学标准（K-5年级）
- 运算与代数思维
- 数与十进制系统
- 分数（3-5年级重点）
- 测量与数据
- 几何

### 英国 NC 关键阶段
- **KS1（1-2年级）**：读写能力、基础算术
- **KS2（3-6年级）**：学科分化、写作深度提升

### 日本小学教育概要
- 国语（语文）、算数、理科、社会科、生活科
- 重视汉字学习（学年汉字配当表）
- 算术系统性极强
