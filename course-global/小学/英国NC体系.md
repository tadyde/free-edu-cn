# 英国国家课程（NC）体系概述

> National Curriculum，英国K-12教育国家标准，分四个关键阶段（Key Stage）。

## 📚 小学阶段：KS1与KS2

### KS1（Key Stage 1）：1-2年级（约6-7岁）
**核心科目**：
- 英语（语文）：自然拼读（Phonics）、阅读流畅度、基础写作
- 数学：100以内数、加减法、基础乘除概念、形状与测量
- 科学：基础科学探究（植物/动物/材料）
- 其他：艺术设计、计算机、设计与技术、体育

**英国特色**：
- 自然拼读法（Phonics）系统教学（类似拼音）
- SATs考试（Standard Assessment Tests）：2年级末

### KS2（Key Stage 2）：3-6年级（约7-11岁）
**核心科目**：
- 英语：阅读理解深度提升（文学作品+信息文本）、正式写作、长篇写作
- 数学：乘除法精算、分数与小数、几何面积、图表
- 科学：实验设计与科学探究（生物/化学/物理入门）
- 其他：历史、地理、宗教教育、 computing、艺术、体育

**KS2 SATs**：6年级末全国统考（英语+数学）

## 🏫 英国小学类型

| 类型 | 说明 |
|------|------|
| 公立小学（Maintained） | 免费，遵循NC标准 |
| 预备学校（Prep School） | 私立，为顶尖中学备考 |
| Academy | 自主管理，可偏离NC标准 |
| Free School | 社区创办，部分自主课程 |

## 💡 与中国小学对比

| 维度 | 英国KS2 | 中国1-6年级 |
|------|---------|------------|
| 学科分化 | 3年级开始明显 | 3年级开始英语，初中才开始物化 |
| 科学教育 | 独立科目，系统性强 | 小学科学课较浅 |
| 写作要求 | 重视创意写作与正式文体 | 侧重看图写话与记叙文 |
| 评估方式 | 过程性评估为主，终考少 | 单元测验频繁 |
| 编程教育 | KS2必修Computing | 近年引入信息技术 |

## 📌 申请英国中学参考

英国顶尖私立中学（Year 7入学约11岁）通常需要：
- 入学测试（英语+数学+逻辑）
- 面试
- 了解NC KS2内容有助于备考
