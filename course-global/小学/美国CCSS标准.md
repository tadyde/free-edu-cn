# 美国CCSS小学标准概述

> Common Core State Standards（共同核心州立标准），美国K-12教育通用标准。

## 📖 CCSS英语标准（K-5年级）

### 阅读标准
- **文学作品**：故事结构、人物分析、主题探究
- **信息文本**：主要观点、细节支撑、文本结构
- **基础阶段**：字母识别、音位意识、流利度

### 写作标准
- **类型**：叙述文（个人故事）、信息文（说明事实）、议论文（表达观点）
- **过程**：计划→起草→修改→编辑→发布
- **研究**：短篇研究报告写作

### 核心特点
1. 强调**文本证据**（text evidence）：所有答案必须引用原文
2. 批判性思维优先于记忆
3. 学术词汇（academic vocabulary）积累贯穿始终

### 关键差异（中国 vs 美国）
- 美国小学三年级才正式引入**分数**，中国二年级已接触简单分数
- 美国CCSS对**代数思维**的铺垫更系统（用字母表示未知数早于中国）
- 几何部分美国更强调**空间推理**，中国更强调公式计算
