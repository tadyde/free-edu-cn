# 🌍 初中国际课程

> 收录主流国际初中课程体系：IB MYP、美国CCSS、英国GCSE入门、日本中学科目体系。

## 📚 课程体系一览

| 课程 | 国家 | 特点 | 适读年龄 |
|------|------|------|----------|
| IB MYP | 全球 | 跨学科探究，批判性思维 | 11-16岁 |
| CCSS | 美国 | 州通用标准，学术扎实 | 11-14岁 |
| GCSE | 英国 | 中考替代科目考试 | 14-16岁 |
| 日本中学 | 日本 | 学科分化细，国数英核心 | 12-15岁 |

## 📋 各体系简述

### 🇺🇸 美国初中（6-8年级）
- 紧随CCSS标准
- 6年级开始代数入门（Pre-Algebra）
- 7-8年级Algebra 1
- Science：生命科学/地球科学/物理

### 🇬🇧 英国初中（Year 7-11）
- Year 7-9：KS3，基础学科探索
- Year 10-11：GCSE考试科目（通常选5-8门）
- 必修：英语、数学、科学
- 选修：历史/地理/二外/艺术等

### 🇯🇵 日本中学（中学1-3年）
- 国语/数学/英语/理科/社会/音楽/美术/保健体育/技术/家庭科
- 1-2年级英语为必修
- 中学3年面临「高校受検」（高中入学考试）

### 🌍 IB MYP
- 5个学科组：语言与文学/语言习得/个人与社会/科学/数学/艺术/设计/体育与健康教育
- 贯穿性主题：身份/社区、时空方位、个人表达、科学技术、公平与发展
- 强调ATL技能（学习方法）：研究/思考/沟通/社交/自我管理
