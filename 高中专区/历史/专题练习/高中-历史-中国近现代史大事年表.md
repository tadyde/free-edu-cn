# 高中历史：中国近现代史大事年表

## 旧民主主义革命（1840-1919）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1840 | 鸦片战争 | 中国开始沦为半殖民地半封建社会 |
| 1842 | 《南京条约》 | 第一个不平等条约 |
| 1851 | 太平天国运动 | 农民战争最高峰 |
| 1856 | 第二次鸦片战争 | 半殖民地化加深 |
| 1894 | 甲午中日战争 | 民族危机加深 |
| 1895 | 《马关条约》 | 割台湾、赔巨款 |
| 1898 | 戊戌变法 | 资产阶级改良尝试 |
| 1900 | 八国联军侵华 | 完全沦为半殖民地半封建社会 |
| 1901 | 《辛丑条约》 | 赔款4.5亿两 |
| 1911 | 辛亥革命 | 推翻封建帝制 |
| 1912 | 中华民国成立 | 亚洲第一个共和国 |
