# 高中议论文写作指导

## 基本结构

### 五段式结构
1. **引论**：引出话题，提出论点
2. **本论一**：分论点1 + 论据 + 分析
3. **本论二**：分论点2 + 论据 + 分析
4. **本论三**：分论点3 + 论据 + 分析（可省略或改为驳论）
5. **结论**：总结升华，呼应开头

## 开头方法

### 1. 引用式
引用名言、诗句、事例引出论点。

> "天下兴亡，匹夫有责。"顾炎武的这句话穿越三百年的时光，至今仍在叩问每一个中国人的心灵。责任，是民族之根，是国家之魂。

### 2. 对比式
正反对比，形成反差，引出论点。

> 有人选择安逸，在舒适区中蹉跎岁月；有人选择挑战，在荆棘路上砥砺前行。人生的分水岭，往往就在于一个"选"字。

### 3. 设问式
提出问题，引发思考，亮出论点。

> 什么是真正的勇敢？是蛮力之下的无所畏惧，还是明知艰险仍毅然前行？我以为，真正的勇敢，是后一种。

### 4. 叙事式
简述事例，自然过渡到论点。

## 分论点设置

### 并列式
分论点之间是并列关系，从不同角度论证中心论点。

论点：读书使人充实
- 分论点1：读书充实人的知识
- 分论点2：读书充实人的精神
- 分论点3：读书充实人的灵魂

### 递进式
分论点之间是递进关系，层层深入。

论点：自律成就人生
- 分论点1：自律是约束自我的能力（是什么）
- 分论点2：自律是克服惰性的利器（为什么）
- 分论点3：自律需要长期坚持和科学方法（怎么做）

### 对比式
正反对比论证。

## 论据使用

### 事实论据
- 历史事件、新闻时事、生活实例
- 要求：真实、典型、新颖
- 写法：简述事实 + 分析与论点的关系

### 道理论据
- 名言警句、科学原理、公认的道理
- 要求：准确、权威、相关
- 写法：引用 + 解释 + 联系论点

## 结尾方法

### 1. 总结式
重申论点，强调核心观点。

### 2. 号召式
发出号召，激励行动。

### 3. 引用式
引用名言，升华主题。

### 4. 展望式
展望未来，寄托希望。

## 常见问题

1. **论点不明确**：开头含糊其辞，读者找不到论点 → 第一段末尾必须亮明论点
2. **论据堆砌**：罗列事例却不分析 → 每个论据后必须分析"为什么"能证明论点
3. **分论点重复**：三个分论点说的都是同一件事 → 用并列/递进/对比确保层次分明
4. **语言平淡**：全是大白话 → 适当使用比喻、排比等修辞手法

---

> 议论文的核心是"说服"——用逻辑和证据让读者认同你的观点。结构是骨架，论据是血肉，语言是外衣，思想是灵魂。
