# 高中物理：力学综合

## 一、牛顿三定律

### 第一定律（惯性定律）
物体不受外力时保持静止或匀速直线运动

### 第二定律
F = ma

### 第三定律
作用力与反作用力大小相等、方向相反、同一直线

## 二、受力分析步骤

1. 确定研究对象
2. 画重力G=mg（竖直向下）
3. 画弹力（接触面法线方向）
4. 画摩擦力（与相对运动趋势相反）
5. 画其他力（电场力、磁场力等）

## 三、运动学公式

v = v0 + at
x = v0*t + 0.5*a*t^2
v^2 - v0^2 = 2ax

## 四、功与能量

- 功：W = F*x*cos(theta)
- 动能：Ek = 0.5*m*v^2
- 势能：Ep = mgh
- 动能定理：W合 = Ek2 - Ek1
- 机械能守恒：Ek1+Ep1 = Ek2+Ep2（只有重力做功时）

## 五、动量

- 动量：p = mv
- 冲量：I = Ft
- 动量定理：Ft = p2 - p1
- 动量守恒：m1v1+m2v2 = m1v1'+m2v2'（合外力为零时）

## 六、常见题型

1. 斜面问题：沿斜面方向分解力
2. 连接体问题：整体法+隔离法
3. 圆周运动：向心力F=mv^2/r
4. 天体运动：GMm/r^2=mv^2/r
