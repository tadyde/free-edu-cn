# 高中物理：电磁学综合

## 一、静电场

### 库仑定律
F = k*q1*q2/r^2（k=9*10^9 N*m^2/C^2）

### 电场强度
E = F/q = kQ/r^2（点电荷）
匀强电场：E = U/d

### 电势与电势差
Uab = phi_a - phi_b
Wab = q*Uab

### 电容
C = Q/U = epsilon*S/(4*pi*k*d)

## 二、恒定电流

### 闭合电路欧姆定律
I = E/(R+r)
路端电压：U = E - Ir

### 电功率
P = UI = I^2R = U^2/R

### 电源功率
P总 = EI
P输出 = EI - I^2r

## 三、磁场

### 安培力
F = BIL（垂直时）

### 洛伦兹力
F = qvB（垂直时）
方向判断：左手定则

### 带电粒子在磁场中做圆周运动
qvB = mv^2/r
r = mv/(qB)
T = 2*pi*m/(qB)

## 四、电磁感应

### 法拉第定律
E = n*delta_phi/delta_t

### 导体切割磁感线
E = BLv（垂直切割时）

### 楞次定律
感应电流的磁场总是阻碍引起感应电流的磁通量的变化

## 五、交变电流

### 正弦交流电
e = Emax*sin(omega*t)
Emax = nBS*omega

### 有效值
U = Umax/sqrt(2)
I = Imax/sqrt(2)

### 变压器
U1/U2 = n1/n2
P1 = P2（理想变压器）
