# 高中物理电学实验总结

> 高考必考5个电学实验

## 实验一：测定金属的电阻率

### 原理
R = ρL/S，测出R、L、S即可求ρ

### 器材选择
- 电流表内外接法判断：
  - Rx >> RA：内接法（电压表分流小）
  - Rx << RV：外接法（电流表分压小）
  - 临界值：Rx = 根号(RA*RV)，大于内接，小于外接

- 滑动变阻器接法：
  - 要求电压从0开始调节：分压式
  - 不要求从0开始且Rx与R0相近：限流式

### 注意事项
- 电流表量程选择
- 多次测量取平均
- 螺旋测微器读数

## 实验二：描绘小灯泡的伏安特性曲线

### 要点
- 必须用分压式接法（需要从0开始调电压）
- 小灯泡电阻较小，用电流表外接法
- 描出的曲线是曲线（电阻随温度变化）
- 曲线斜率减小——电阻增大（温度升高）

## 实验三：测定电源电动势和内阻

### 原理
E = U + Ir

### 方法
1. 伏安法：改变外电阻，测多组U和I
   - U-I图像：斜率=|-r|，纵截距=E
2. 安阻法：已知R，测I
3. 伏阻法：已知R，测U

### 注意
- 电流表一般用内接法（因为r一般较小）
- 误差分析：系统误差来源

## 实验四：练习使用多用电表

### 使用方法
1. 机械调零
2. 选挡位
3. 测量
4. 读数（注意倍率）
5. 用毕拨到OFF或交流最高挡

### 测电阻
- 选挡→欧姆调零→测量→读数
- 换挡后必须重新欧姆调零
- 指针在中值附近最准确

## 实验五：传感器的简单使用

### 常见传感器
- 光敏电阻：光照增强→电阻减小
- 热敏电阻：温度升高→电阻减小（负温度系数）
- 金属热电阻：温度升高→电阻增大

### 应用
- 光控开关
- 温度报警器
