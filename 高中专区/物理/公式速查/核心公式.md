# 高中物理核心公式

## 力学

### 运动学
- v = v0 + at
- s = v0*t + 0.5*a*t^2
- v^2 - v0^2 = 2as
- 自由落体：v = gt, h = 0.5*g*t^2

### 牛顿运动定律
- F = ma（牛顿第二定律）
- F合 = 0 等价于 匀速或静止

### 曲线运动
- 平抛：x = v0*t, y = 0.5*g*t^2
- 圆周运动：v = 2*Pi*r/T, a向心 = v^2/r = 4*Pi^2*r/T^2
- 向心力：F向心 = m*v^2/r

### 万有引力
- F = G*m1*m2/r^2
- 第一宇宙速度：v1 = 根号(gR) = 7.9 km/s
- 轨道速度：v = 根号(GM/r)

### 功和能
- W = F*s*cos(θ)
- P = W/t = F*v
- 动能：Ek = 0.5*m*v^2
- 动能定理：W合 = Ek2 - Ek1
- 重力势能：Ep = m*g*h
- 机械能守恒：Ek1+Ep1 = Ek2+Ep2

## 电磁学

### 静电场
- 库仑定律：F = k*q1*q2/r^2
- 电场强度：E = F/q
- 电势能：Ep = q*φ
- 电势差：U = W/q
- 平行板电容：C = εS/(4πkd)

### 恒定电流
- 欧姆定律：I = U/R
- 串联：R总 = R1+R2
- 并联：1/R总 = 1/R1+1/R2
- 电功：W = UIt
- 电功率：P = UI
- 闭合电路：E = U外 + U内 = I*(R+r)

### 磁场
- 安培力：F = BIL*sin(θ)
- 洛伦兹力：f = qvB*sin(θ)
- 带电粒子在磁场中做圆周运动：r = mv/(qB)

### 电磁感应
- 法拉第定律：E = n*ΔΦ/Δt
- 导体切割：E = BLv
