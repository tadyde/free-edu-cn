# 高中英语虚拟语气

## 一、if条件句中的虚拟

| 情况 | if从句 | 主句 |
|------|--------|------|
| 与现在相反 | did/were | would/should/could/might + do |
| 与过去相反 | had done | would/should/could/might + have done |
| 与将来相反 | did/were to do/should do | would/should/could/might + do |

### 例句
If I were you, I would study harder.
If I had known, I would have helped you.
If it should rain tomorrow, we would stay home.

## 二、省略if的倒装

If I had arrived earlier → Had I arrived earlier
If I were you → Were I you
If it should rain → Should it rain

## 三、wish后的虚拟

- wish + did/were（与现在相反）
- wish + had done（与过去相反）
- wish + would/could do（与将来相反）

I wish I were a bird.
I wish I had studied harder.
I wish I could fly.

## 四、as if/though

与wish用法相同：
He talks as if he knew everything.
He looked as if he had seen a ghost.

## 五、名词性从句中的虚拟

### insist/demand/suggest/order/advise等后
should + do（should可省略）

I suggest that he (should) go at once.
The doctor ordered that the patient (should) rest.

### 形容词important/necessary/essential等后
It is important that he (should) attend the meeting.

## 六、其他虚拟

### It is (high) time that
did / should do
It is high time that we went home.

### would rather
would rather + did
I would rather you came tomorrow.

### but for / without
= if it were not for / had not been for
But for your help, I would have failed.
