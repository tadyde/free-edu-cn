# 高中生物核心概念速查

## 细胞

### 细胞膜
- 功能：保护、控制物质进出、细胞间信息交流
- 结构：磷脂双分子层 + 蛋白质（流动镶嵌模型）
- 特性：选择透过性

### 细胞器
- 线粒体：有氧呼吸主要场所，"动力车间"
- 叶绿体：光合作用场所
- 核糖体：蛋白质合成场所
- 内质网：蛋白质加工运输（粗面）、脂质合成（滑面）
- 高尔基体：蛋白质进一步加工、分类、包装运输
- 溶酶体：细胞内消化
- 液泡：调节细胞内环境（植物细胞）

### 细胞核
- 结构：核膜（双层膜）、核仁、染色质、核孔
- 功能：遗传信息库，细胞代谢和遗传的控制中心

## 光合作用与呼吸作用

### 光合作用
- 场所：叶绿体
- 总反应：6CO2 + 12H2O --光能--> C6H12O6 + 6O2 + 6H2O
- 光反应：H2O分解，产生O2、[H]和ATP
- 暗反应：CO2固定（C3），C3还原（需[H]和ATP）

### 细胞呼吸
- 有氧呼吸：C6H12O6 + 6O2 + 6H2O --> 6CO2 + 12H2O + 能量
  - 第一阶段：细胞质基质（糖酵解）
  - 第二阶段：线粒体基质
  - 第三阶段：线粒体内膜
- 无氧呼吸：
  - 酒精发酵：C6H12O6 --> 2C2H5OH + 2CO2 + 能量
  - 乳酸发酵：C6H12O6 --> 2C3H6O3 + 能量

## 遗传

### DNA
- 双螺旋结构（沃森和克里克）
- 碱基互补配对：A-T, G-C
- 半保留复制

### 中心法则
DNA --> DNA（复制）
DNA --> RNA（转录）
RNA --> 蛋白质（翻译）
RNA --> RNA（RNA复制，某些病毒）
RNA --> DNA（逆转录，某些病毒）

### 遗传规律
- 分离定律：等位基因随同源染色体分离
- 自由组合定律：非同源染色体上的非等位基因自由组合

### 伴性遗传
- X染色体显性遗传：女性患者多
- X染色体隐性遗传：男性患者多（色盲、血友病）
- Y染色体遗传：只在男性中传递
