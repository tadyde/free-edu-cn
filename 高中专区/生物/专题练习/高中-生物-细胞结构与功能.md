# 高中生物：细胞结构与功能

## 一、细胞膜

### 成分
磷脂双分子层 + 蛋白质 + 糖类

### 功能
1. 将细胞与外界环境分隔开
2. 控制物质进出细胞
3. 进行细胞间的信息传递

### 流动镶嵌模型
- 磷脂双分子层：基本骨架，具有流动性
- 蛋白质：镶、嵌、贯穿
- 糖蛋白：细胞识别

## 二、细胞器

### 双层膜
| 细胞器 | 功能 |
|--------|------|
| 线粒体 | 有氧呼吸主要场所，动力车间 |
| 叶绿体 | 光合作用场所（植物特有） |

### 单层膜
| 细胞器 | 功能 |
|--------|------|
| 内质网 | 蛋白质加工运输、脂质合成 |
| 高尔基体 | 蛋白质进一步加工分类包装 |
| 液泡 | 调节渗透压（植物特有） |
| 溶酶体 | 细胞内消化 |

### 无膜
| 细胞器 | 功能 |
|--------|------|
| 核糖体 | 蛋白质合成场所 |
| 中心体 | 参与细胞分裂（动物、低等植物） |

## 三、细胞核

### 结构
- 核膜：双层膜，有核孔
- 核仁：rRNA合成场所
- 染色质：DNA+蛋白质

### 功能
遗传信息库，细胞代谢和遗传的控制中心

## 四、原核细胞与真核细胞

| 特征 | 原核细胞 | 真核细胞 |
|------|---------|---------|
| 细胞核 | 无核膜 | 有核膜 |
| DNA | 拟核（裸露） | 染色体（含蛋白质） |
| 细胞器 | 只有核糖体 | 多种 |
| 举例 | 细菌、蓝藻 | 动植物细胞 |

## 五、分泌蛋白的合成和运输

核糖体合成 → 内质网加工运输 → 高尔基体进一步加工 → 细胞膜分泌

整个过程需要线粒体供能。
