# 高中生物：遗传与变异

## 一、孟德尔遗传定律

### 分离定律
- 控制一对相对性状的等位基因在减数分裂时分离
- 杂合子Aa自交：AA:Aa:aa = 1:2:1
- 表现型比 = 3:1（完全显性）

### 自由组合定律
- 控制不同性状的基因自由组合
- AaBb自交：表现型比 = 9:3:3:1

## 二、基因在染色体上

- 萨顿假说：基因和染色体行为存在平行关系
- 摩尔根实验：果蝇白眼突变证明基因在染色体上

## 三、伴性遗传

### X染色体显性遗传
- 女性患者多于男性
- 男患者的女儿都患病

### X染色体隐性遗传
- 男性患者多于女性
- 母亲患病儿子必患病
- 交叉遗传

## 四、变异类型

| 类型 | 是否可遗传 | 举例 |
|------|-----------|------|
| 基因突变 | 是 | 镰刀型细胞贫血症 |
| 基因重组 | 是 | 杂交育种 |
| 染色体变异 | 是 | 21三体综合征 |
| 环境引起 | 否 | 同一品种不同环境表现不同 |

## 五、育种方法

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 杂交育种 | 基因重组 | 操作简单 | 周期长 |
| 诱变育种 | 基因突变 | 提高变异频率 | 盲目性大 |
| 单倍体育种 | 染色体变异 | 明显缩短育种年限 | 技术复杂 |
| 多倍体育种 | 染色体变异 | 器官大 | 发育延迟 |
| 基因工程育种 | 基因重组 | 定向改造 | 安全争议 |
