# 高中化学：氧化还原反应

## 一、基本概念

- 氧化反应：失去电子，化合价升高
- 还原反应：得到电子，化合价降低
- 氧化剂：得到电子的物质（化合价降低），具有氧化性
- 还原剂：失去电子的物质（化合价升高），具有还原性

## 二、口诀

**升失氧还降得还氧**

- 化合价升高 → 失电子 → 氧化反应 → 还原剂
- 化合价降低 → 得电子 → 还原反应 → 氧化剂

## 三、常见氧化剂

| 强弱 | 氧化剂 | 还原产物 |
|------|--------|----------|
| 强 | KMnO4(H+) | Mn2+ |
| 强 | Cl2 | Cl- |
| 较强 | 浓H2SO4 | SO2 |
| 较强 | HNO3(浓) | NO2 |
| 较强 | HNO3(稀) | NO |
| 中等 | Fe3+ | Fe2+ |
| 弱 | I2 | I- |

## 四、常见还原剂

| 强弱 | 还原剂 | 氧化产物 |
|------|--------|----------|
| 强 | 金属（Na、Mg、Al等） | 金属阳离子 |
| 较强 | H2S / S2- | S |
| 较强 | I- | I2 |
| 中等 | Fe2+ | Fe3+ |
| 弱 | SO3 2- | SO4 2- |

## 五、氧化还原反应配平

### 步骤
1. 标化合价
2. 找化合价变化
3. 使升降总数相等（最小公倍数法）
4. 用观察法配平其他物质
5. 检查原子守恒和电荷守恒

### 例题
Cu + HNO3(稀) → Cu(NO3)2 + NO + H2O

Cu: 0→+2, 升2
N: +5→+2, 降3

最小公倍数6: 3Cu * 升2 = 6, 2HNO3 * 降3 = 6

3Cu + 8HNO3(稀) → 3Cu(NO3)2 + 2NO + 4H2O

## 六、原电池与电解池

### 原电池
- 负极：氧化反应（失电子）
- 正极：还原反应（得电子）
- 电子：负极→外电路→正极

### 电解池
- 阳极：氧化反应（与电源正极相连）
- 阴极：还原反应（与电源负极相连）
