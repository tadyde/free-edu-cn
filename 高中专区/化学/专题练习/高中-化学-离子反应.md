# 高中化学：离子反应

## 一、电解质与非电解质

- 电解质：在水溶液或熔融态能导电的化合物（酸、碱、盐）
- 非电解质：在水溶液和熔融态都不能导电的化合物（蔗糖、酒精）

## 二、电离方程式

- 强酸：HCl = H+ + Cl-
- 弱酸：CH3COOH <-> H+ + CH3COO-
- 强碱：NaOH = Na+ + OH-
- 弱碱：NH3*H2O <-> NH4+ + OH-
- 盐：NaCl = Na+ + Cl-

## 三、离子反应发生条件

1. 生成沉淀
2. 生成气体
3. 生成弱电解质（水、弱酸、弱碱）
4. 发生氧化还原反应

## 四、常见沉淀

| 颜色 | 物质 |
|------|------|
| 白色 | BaSO4 AgCl Mg(OH)2 Al(OH)3 |
| 蓝色 | Cu(OH)2 |
| 红褐色 | Fe(OH)3 |
| 黑色 | CuS FeS |
| 黄色 | AgI Ag3PO4 |

## 五、离子共存判断

不能大量共存的离子对：
- 生成沉淀：Ba2+与SO4 2-，Ag+与Cl-
- 生成气体：H+与CO3 2-，H+与SO3 2-
- 生成弱电解质：H+与OH-，H+与CH3COO-
- 氧化还原：Fe3+与I-，MnO4-与Fe2+（酸性条件）
- 双水解：Al3+与AlO2-，Fe3+与CO3 2-
