# 高中化学核心公式

## 物质的量

- n = m/M（物质的量 = 质量/摩尔质量）
- n = N/NA（物质的量 = 粒子数/阿伏加德罗常数）
- n = V/Vm（气体物质的量 = 体积/摩尔体积）
- 标准状况下 Vm = 22.4 L/mol
- NA = 6.02 x 10^23 /mol
- c = n/V（物质的量浓度 = 物质的量/溶液体积）

## 氧化还原反应

- 氧化剂 + 还原剂 = 还原产物 + 氧化产物
- 氧化剂得电子，化合价降低，被还原
- 还原剂失电子，化合价升高，被氧化
- 电子转移守恒：失电子总数 = 得电子总数

## 化学反应速率

- v = Δc/Δt
- 影响因素：浓度、温度、催化剂、接触面积

## 化学平衡

- 平衡常数 K = [C]^c[D]^d / [A]^a[B]^b
- K只与温度有关
- 浓度商 Q 与 K 比较：
  - Q < K：正方向移动
  - Q = K：平衡
  - Q > K：逆方向移动

## 弱电解质电离

- 弱酸电离：HA = H+ + A-
- Ka = [H+][A-]/[HA]
- 水的离子积：Kw = [H+][OH-] = 1.0 x 10^-14 (25度C)

## pH计算

- pH = -lg[H+]
- pH + pOH = 14 (25度C)

## 电化学

### 原电池
- 负极：氧化反应（失电子）
- 正极：还原反应（得电子）
- 电子流向：负极 -> 外电路 -> 正极

### 电解池
- 阳极：氧化反应
- 阴极：还原反应
- 电子流向：电源负极 -> 阴极 -> 阳极 -> 电源正极
