## 有机化合物的分类

### 按碳骨架分类
- 链状化合物（脂肪族）
- 环状化合物（脂环族、芳香族）

### 按官能团分类
| 官能团 | 名称 | 代表物 |
|--------|------|--------|
| C=C | 碳碳双键 | 乙烯 |
| C≡C | 碳碳三键 | 乙炔 |
| -OH | 羟基 | 乙醇 |
| -CHO | 醛基 | 乙醛 |
| -COOH | 羧基 | 乙酸 |
| -COO- | 酯基 | 乙酸乙酯 |

## 烃

### 烷烃（CnH2n+2）
- 甲烷 CH4：正四面体，键角109度28分
- 取代反应（与卤素单质在光照条件下）
- 燃烧：CnH2n+2 + (3n+1)/2 O2 --> nCO2 + (n+1)H2O

### 烯烃（CnH2n）
- 乙烯 CH2=CH2：平面结构，6原子共面
- 加成反应（与Br2、H2、HCl、H2O）
- 加聚反应：nCH2=CH2 --> [-CH2-CH2-]n

### 炔烃（CnH2n-2）
- 乙炔 CH≡CH：直线形，4原子共线

### 苯（C6H6）
- 平面正六边形，12个原子共面
- 特殊结构：碳碳键完全等价
- 取代反应（溴代、硝化）
- 加成反应（与H2）

## 烃的含氧衍生物

### 醇（-OH）
- 乙醇 CH3CH2OH
- 与Na反应产生H2
- 催化氧化：2CH3CH2OH + O2 --Cu,加热--> 2CH3CHO + 2H2O
- 酯化反应（与羧酸）

### 醛（-CHO）
- 乙醛 CH3CHO
- 银镜反应（检验醛基）
- 与新制Cu(OH)2反应（砖红色沉淀）

### 羧酸（-COOH）
- 乙酸 CH3COOH（醋酸）
- 弱酸性，比碳酸强
- 酯化反应：CH3COOH + C2H5OH --浓H2SO4,加热--> CH3COOC2H5 + H2O

### 酯（-COO-）
- 水解反应（酸性/碱性条件下）

## 同分异构体

### 类型
- 碳链异构
- 位置异构
- 官能团异构

### 书写方法
1. 先写碳链异构
2. 再写位置异构
3. 最后检查官能团异构

## 常见反应类型

| 反应类型 | 特点 |
|----------|------|
| 取代反应 | 有上有下 |
| 加成反应 | 只上不下 |
| 消去反应 | 只下不上 |
| 氧化反应 | 加氧或去氢 |
| 还原反应 | 加氢或去氧 |
| 加聚反应 | 双键变单键，连成长链 |
| 缩聚反应 | 小分子脱出，连成长链 |
