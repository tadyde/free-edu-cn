# 高中数学：三角函数公式汇总

## 基本关系

sin^2(x) + cos^2(x) = 1
tan(x) = sin(x)/cos(x)
1 + tan^2(x) = sec^2(x)
1 + cot^2(x) = csc^2(x)

## 诱导公式（口诀：奇变偶不变，符号看象限）

sin(-x) = -sin(x)
cos(-x) = cos(x)
sin(pi-x) = sin(x)
cos(pi-x) = -cos(x)
sin(pi+x) = -sin(x)
cos(pi+x) = -cos(x)

## 和差公式

sin(a+b) = sin(a)cos(b) + cos(a)sin(b)
sin(a-b) = sin(a)cos(b) - cos(a)sin(b)
cos(a+b) = cos(a)cos(b) - sin(a)sin(b)
cos(a-b) = cos(a)cos(b) + sin(a)sin(b)

## 二倍角公式

sin(2x) = 2sin(x)cos(x)
cos(2x) = cos^2(x) - sin^2(x) = 2cos^2(x) - 1 = 1 - 2sin^2(x)
tan(2x) = 2tan(x)/(1-tan^2(x))

## 半角公式

sin(x/2) = +/- sqrt((1-cos(x))/2)
cos(x/2) = +/- sqrt((1+cos(x))/2)

## 辅助角公式

a*sin(x) + b*cos(x) = sqrt(a^2+b^2) * sin(x+phi)
其中 tan(phi) = b/a
