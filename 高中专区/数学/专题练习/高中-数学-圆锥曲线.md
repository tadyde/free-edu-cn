# 高中数学：圆锥曲线

## 一、椭圆

### 标准方程
x^2/a^2 + y^2/b^2 = 1（a>b>0，焦点在x轴）

### 关系
c^2 = a^2 - b^2
e = c/a（离心率，0<e<1）

### 焦点
F1(-c,0), F2(c,0)

### 性质
|PF1| + |PF2| = 2a

## 二、双曲线

### 标准方程
x^2/a^2 - y^2/b^2 = 1（焦点在x轴）

### 关系
c^2 = a^2 + b^2
e = c/a（离心率，e>1）

### 渐近线
y = +/- (b/a)x

## 三、抛物线

### 标准方程
y^2 = 2px（焦点在x轴正半轴，p>0）

### 焦点
F(p/2, 0)

### 准线
x = -p/2

### 性质
|PF| = |x_P| + p/2

## 四、解题技巧

1. 设而不求法：设交点坐标但不具体求出
2. 韦达定理：联立方程后用根与系数关系
3. 焦点三角形：利用椭圆/双曲线定义
4. 点差法：中点弦问题
5. 参数方程法：设参数化简运算
