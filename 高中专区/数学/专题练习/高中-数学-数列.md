# 高中数学：数列

## 一、等差数列

### 定义
a(n+1) - an = d（常数d为公差）

### 通项公式
an = a1 + (n-1)d

### 前n项和
Sn = n*a1 + n(n-1)d/2 = n(a1+an)/2

### 性质
- am + an = ap + aq（m+n=p+q时）
- S2n-1 = (2n-1)*an

## 二、等比数列

### 定义
a(n+1)/an = q（常数q为公比，q不等于0）

### 通项公式
an = a1 * q^(n-1)

### 前n项和
q=1时：Sn = n*a1
q不等于1时：Sn = a1(1-q^n)/(1-q)

### 性质
- am * an = ap * aq（m+n=p+q时）

## 三、求通项方法

1. 已知Sn求an：an = Sn - S(n-1)（n>=2）
2. an+1/an = f(n)型：累乘法
3. an+1 - an = f(n)型：累加法
4. an+1 = p*an + q型：待定系数法构造等比数列

## 四、求和方法

1. 公式法（等差/等比直接用公式）
2. 分组求和法
3. 裂项相消法
4. 错位相减法（等差*等比型）
5. 倒序相加法

## 五、常见裂项

1/(n(n+1)) = 1/n - 1/(n+1)
1/((2n-1)(2n+1)) = 1/2 * (1/(2n-1) - 1/(2n+1))
1/(sqrt(n)+sqrt(n+1)) = sqrt(n+1) - sqrt(n)
