# 高中数学：导数与单调性

## 一、导数的定义

f'(x) = lim(h->0) [f(x+h) - f(x)] / h

## 二、基本求导公式

| 函数 | 导数 |
|------|------|
| c（常数） | 0 |
| x^n | nx^(n-1) |
| sin(x) | cos(x) |
| cos(x) | -sin(x) |
| e^x | e^x |
| ln(x) | 1/x |
| a^x | a^x * ln(a) |

## 三、求导法则

- (f+g)' = f' + g'
- (f*g)' = f'*g + f*g'
- (f/g)' = (f'*g - f*g') / g^2
- 链式法则：[f(g(x))]' = f'(g(x)) * g'(x)

## 四、导数与单调性

- f'(x)>0 → f(x)单调递增
- f'(x)<0 → f(x)单调递减
- f'(x)=0 → 可能是极值点

## 五、求单调区间步骤

1. 求f'(x)
2. 解f'(x)>0得增区间
3. 解f'(x)<0得减区间
4. 注意定义域

## 六、例题

**例**：求f(x)=x^3-3x的单调区间。

解：f'(x)=3x^2-3=3(x+1)(x-1)
x<-1时，f'(x)>0，递增
-1<x<1时，f'(x)<0，递减
x>1时，f'(x)>0，递增

单调递增区间：(-无穷,-1)和(1,+无穷)
单调递减区间：(-1,1)
