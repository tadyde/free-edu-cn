# 高中数学核心公式速查

## 集合

| 公式 | 说明 |
|------|------|
| A交集B | 同时属于A和B的元素 |
| A并集B | 属于A或B的元素 |
| 补集CuA | 全集中不属于A的元素 |
| card(A并B) = card(A)+card(B)-card(A交B) | 容斥原理 |

## 基本不等式

- (a+b)/2 大于等于 根号(ab)，a>0, b>0
- 等号条件：a = b

## 指数运算

- a^(m+n) = a^m * a^n
- a^(m-n) = a^m / a^n
- (a^m)^n = a^(mn)
- a^(-n) = 1/a^n

## 对数运算

- log_a(MN) = log_a(M) + log_a(N)
- log_a(M/N) = log_a(M) - log_a(N)
- log_a(M^n) = n * log_a(M)
- 换底公式：log_a(b) = ln(b)/ln(a)

## 三角函数

### 基本关系
- sin^2(a) + cos^2(a) = 1
- tan(a) = sin(a)/cos(a)
- 1 + tan^2(a) = 1/cos^2(a)

### 诱导公式口诀：奇变偶不变，符号看象限
- sin(180-a) = sin(a)
- cos(180-a) = -cos(a)
- sin(90-a) = cos(a)
- cos(90-a) = sin(a)

### 和差公式
- sin(a+b) = sin(a)cos(b) + cos(a)sin(b)
- cos(a+b) = cos(a)cos(b) - sin(a)sin(b)

### 二倍角公式
- sin(2a) = 2sin(a)cos(a)
- cos(2a) = cos^2(a) - sin^2(a) = 2cos^2(a)-1 = 1-2sin^2(a)

## 向量

- 数量积：a*b = |a||b|cos(θ)
- a垂直b 等价于 a*b = 0
- a平行b 等价于 存在λ使得 a = λb

## 数列

### 等差数列
- 通项：a_n = a_1 + (n-1)d
- 前n项和：S_n = n(a_1+a_n)/2

### 等比数列
- 通项：a_n = a_1 * q^(n-1)
- 前n项和：S_n = a_1(1-q^n)/(1-q)，q不等于1

## 导数

### 常用求导公式

| 函数 | 导数 |
|------|------|
| c（常数） | 0 |
| x^n | nx^(n-1) |
| e^x | e^x |
| a^x | a^x * ln(a) |
| lnx | 1/x |
| sinx | cosx |
| cosx | -sinx |

### 运算法则
- (f+g)' = f'+g'
- (fg)' = f'g + fg'
- (f/g)' = (f'g - fg')/g^2
- 链式法则：[f(g(x))]' = f'(g(x)) * g'(x)

## 解析几何

### 直线
- 点斜式：y - y0 = k(x - x0)
- 点到直线距离：d = |Ax0+By0+C|/根号(A^2+B^2)

### 圆
- 标准方程：(x-a)^2 + (y-b)^2 = r^2

### 椭圆
- x^2/a^2 + y^2/b^2 = 1（a>b>0）
- c^2 = a^2 - b^2，离心率 e = c/a

### 双曲线
- x^2/a^2 - y^2/b^2 = 1
- c^2 = a^2 + b^2，渐近线 y = (+/-)(b/a)x

### 抛物线
- y^2 = 2px，焦点(p/2, 0)，准线x = -p/2
