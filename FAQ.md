# 常见问题

## 这个项目是什么？

这是一个面向全国小学和初中学生的免费公益教育资源仓库。所有内容完全免费，无广告，开源共享。

## 资料是免费的吗？

**完全免费。** 本项目采用 CC BY-NC-SA 4.0 许可协议，任何人都可以免费使用和分享。

## 资料内容准确吗？

我们的知识点清单依据教育部《义务教育课程标准（2022年版）》和人民教育出版社统编教材整理。如发现错误，欢迎指正。

## 可以打印这些资料吗？

可以。所有资料都可以免费下载和打印，供个人学习使用。

## 如何贡献资料？

请阅读 [贡献指南](CONTRIBUTING.md)。简单来说：
1. Fork 本仓库
2. 添加或修改内容
3. 提交 Pull Request

## 我不是老师，可以贡献吗？

当然可以！学生、家长、任何愿意分享知识的人都可以贡献。哪怕只是发现一个错别字，也欢迎提 Issue。

## 资料是哪个版本的教材？

主要参考人教版（统编版）教材体系。不同地区使用的教材版本可能不同（如北师大版、苏教版等），但核心知识点是一致的。

## 有高中内容吗？

目前暂无。本项目专注于小学和初中阶段。如果未来有条件，可能会扩展到高中。

## 可以用在商业场景吗？

**不可以。** CC BY-NC-SA 4.0 许可协议明确禁止商业用途。这些资料只能免费用于教育目的。
