# 高中化学

> 匹配**人教版化学教材**，必修+选修体系，反应原理→有机化学→物质结构。

## 📖 核心模块

| 模块 | 内容 | 高考占比 |
|------|------|----------|
| 化学概念 | 物质的量/氧化还原/离子反应 | 约15% |
| 元素化合物 | 金属/非金属性质 | 约15% |
| 反应原理 | 化学反应速率/平衡/水溶液 | 约20% |
| 有机化学 | 烃/官能团/合成路线 | 约15% |
| 物质结构 | 原子结构/周期律/分子结构 | 约15% |
| 实验化学 | 物质的制备/分离/检验 | 约20% |

## 📋 各年级重点

| 年级 | 核心内容 |
|------|----------|
| 高一 | 物质的量、氧化还原、元素化合物 |
| 高二上 | 反应原理（速率/平衡/水溶液）、物质结构 |
| 高二下-高三 | 有机化学、实验专题、综合复习 |

## ⚠️ 关键难点

- **水溶液中的离子平衡**：电离/水解/沉淀溶解，图像题
- **有机合成路线**：反应类型串联、逆向合成思维
- **化学用语**：方程式书写（含过量/先后顺序/条件控制）
