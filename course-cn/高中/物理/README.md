# 高中物理

> 匹配**人教版物理教材**，力学→电磁学→热学→光学→原子物理全面覆盖。

## 📖 核心模块（按高考占比）

| 模块 | 占比 | 核心内容 |
|------|------|----------|
| 力学 | 约40% | 运动学/动力学/能量/动量/机械振动 |
| 电学 | 约35% | 电场/电路/磁场/电磁感应/交流电 |
| 热学 | 约10% | 分子动理论/气体状态方程/热力学定律 |
| 光学 | 约10% | 几何光学/波动光学（干涉/衍射/偏振）|
| 原子物理 | 约5% | 原子结构/原子核/波粒二象性 |

## 📋 各年级重点

| 年级 | 核心任务 |
|------|----------|
| 高一 | 力学基础（运动/力/能量/动量）——高中最难模块 |
| 高二上 | 电学（电场→电路→磁场→电磁感应） |
| 高二下 | 选修：热学/光学/原子物理 |
| 高三 | 综合复习、模型总结、真题训练 |

## ⚠️ 关键难点

- **力学综合**：三大观点（力/能/动量）贯穿始终
- **电磁感应**：楞次定律、法拉第定律、综合分析
- **实验**：打点计时器/验证机械能守恒/测电阻/测电源电动势
