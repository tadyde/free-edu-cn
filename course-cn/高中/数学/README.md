# 高中数学

> 匹配**人教A/B版教材**，函数→几何→概率的系统进阶。

## 📖 核心模块（按高考占比）

| 模块 | 占比 | 核心内容 |
|------|------|----------|
| 函数 | 约35% | 指对幂函数、三角函数、导数 |
| 解析几何 | 约20% | 直线与圆、椭圆/双曲线/抛物线 |
| 立体几何 | 约15% | 空间向量、线面关系 |
| 数列 | 约10% | 等差/等比/求和/递推 |
| 概率统计 | 约10% | 排列组合、二项分布、正态分布 |
| 不等式 | 约10% | 基本不等式、线性规划 |

## 📋 文理差异（全国卷参考）

- **理科**：增加计数原理、排列组合、二项式定理、随机变量分布
- **文科**：侧重统计案例、基础概率
