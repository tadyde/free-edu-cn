# 高中地理

> 匹配**人教版地理教材**，自然地理→人文地理→区域地理→选修模块。

## 📖 核心模块

| 模块 | 核心内容 | 高考占比 |
|------|----------|----------|
| 自然地理 | 地球/大气/水文/地貌/整体性差异性 | 约35% |
| 人文地理 | 人口/城市/农业/工业/交通/环境 | 约30% |
| 区域地理 | 区域分析/可持续发展/中国/世界 | 约25% |
| 选修模块 | 旅游/环境保护/海洋地理 | 约10% |

## ⚠️ 关键难点

- **等值线判读**：等高线/等温线/等压线
- **大气环流**：三圈环流/季风/气候类型
- **区位分析**：农业/工业/城市区位因素的全面性
