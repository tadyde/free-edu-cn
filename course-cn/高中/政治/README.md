# 高中思想政治

> 匹配**部编版思想政治教材**，经济/政治/文化/哲学四模块。

## 📖 核心模块

| 模块 | 核心内容 | 高考占比 |
|------|----------|----------|
| 经济生活 | 消费/生产/分配/交换/市场经济 | 约25% |
| 政治生活 | 公民/政府/政党/国际关系 | 约20% |
| 文化生活 | 文化作用/传承/创新/民族精神 | 约20% |
| 生活与哲学 | 唯物论/辩证法/认识论/历史唯物主义 | 约35% |

## ⚠️ 关键难点

- **哲学模块**：唯物辩证法（联系/发展/矛盾/否定）+ 认识论
- **经济计算**：商品价值量/汇率/恩格尔系数
- **大题答题**：原理+方法论+材料的答题模板
