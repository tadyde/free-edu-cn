# 高中历史

> 匹配**部编版历史教材**，中国古代史+世界史+中国近现代史。

## 📖 三年知识体系

| 模块 | 核心内容 | 重点时期 |
|------|----------|----------|
| 中国古代史 | 政治制度/经济/思想文化 | 秦汉/唐宋/明清 |
| 中国近现代史 | 近代化探索/革命/新中国建设 | 鸦片战争/民国/建国后 |
| 世界古代史 | 希腊罗马/中世纪 | 民主制度起源 |
| 世界近现代史 | 资本主义发展/两次大战/冷战 | 工业革命/十月革命/战后格局 |

## 📋 高考历史备考重点

- **时间线串联**：建立中外历史横向对比框架
- **阶段特征**：每个历史时期的政治/经济/文化特征
- **史料实证**：学会从史料中提取信息、分析问题
