# 高中生物

> 匹配**人教版生物教材**，分子与细胞→遗传与进化→稳态与环境→生物技术。

## 📖 核心模块

| 模块 | 核心内容 | 高考占比 |
|------|----------|----------|
| 分子与细胞 | 细胞结构/代谢（光合/呼吸）/分裂 | 约35% |
| 遗传与变异 | 遗传规律/分子基础/变异育种 | 约30% |
| 稳态与环境 | 神经体液免疫/种群群落/生态系统 | 约25% |
| 生物技术与工程 | 传统发酵/细胞工程/基因工程 | 约10% |

## ⚠️ 关键难点

- **遗传系谱图分析**：显隐判断、概率计算
- **细胞呼吸与光合作用综合**：C3/C4途径、净光合/总光合
- **基因工程**：PCR/凝胶电泳/限制酶/载体构建
