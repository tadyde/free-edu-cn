# 高中语文

> 匹配**部编版语文教材**，必修+选择性必修+选修体系。

## 📖 知识体系

| 模块 | 占比（高考） | 重点 |
|------|-------------|------|
| 现代文阅读 | 约25% | 信息性阅读（论述/实用/小说/散文） |
| 古诗文阅读 | 约35% | 文言文实词虚词、古代诗歌鉴赏 |
| 语言文字运用 | 约10% | 字音字形、成语、病句、衔接 |
| 写作 | 约30% | 议论文/记叙文（近年以议论文为主） |

## 📋 高中三年重点

| 年级 | 核心能力 |
|------|----------|
| 高一 | 现代文阅读方法、议论文入门、文言文积累 |
| 高二 | 深度文本分析、文学鉴赏、作文提升 |
| 高三 | 高考真题训练、速度与准确率并重 |
