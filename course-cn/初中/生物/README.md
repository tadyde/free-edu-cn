# 初中生物

> 匹配**人教版/北师大版生物教材**，7-8年级生物学科学习。

## 📖 三年知识体系

| 年级 | 内容重点 |
|------|----------|
| 7年级上 | 生物与环境、植物体的结构层次、动物体的结构层次 |
| 7年级下 | 人体的营养/呼吸/物质运输/废物的排出 |
| 8年级上 | 动物的运动与行为、动物在生物圈中的作用 |
| 8年级下 | 生殖与发育、遗传与变异、生物进化 |
| 总复习 | 生态系统、生物圈、微生物 |

## 📋 中考生物分值分布（参考）

| 模块 | 占比 |
|------|------|
| 生物与环境（生态学） | 约15% |
| 植物学（结构/生理） | 约25% |
| 人体生理（消化/呼吸/循环等） | 约30% |
| 动物学与微生物 | 约15% |
| 遗传与进化 | 约15% |
