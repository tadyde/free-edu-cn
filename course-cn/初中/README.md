## 🎯 中考备考建议

1. **一轮巩固**：通读知识点清单，建立学科框架
2. **二轮专项**：专题练习突破薄弱章节
3. **三轮冲刺**：真题模拟，查漏补缺

## 📋 各年级重点

| 年级 | 核心任务 |
|------|----------|
| 7年级 | 适应初中节奏，打好语数英基础 |
| 8年级 | 新增物理，数学难度提升（两极分化关键期） |
| 9年级 | 中考总复习，冲刺目标高中 |
