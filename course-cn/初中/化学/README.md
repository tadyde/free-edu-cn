# 初中化学

> 匹配**人教版化学教材**，9年级开设，系统入门化学世界。

## 📖 三年知识体系（9年级集中学习）

| 单元 | 核心内容 |
|------|----------|
| 走进化学世界 | 实验仪器、药品取用、加热、连接装置 |
| 我们周围的空气 | 空气成分、氧气性质与制取 |
| 物质构成的奥秘 | 分子、原子、离子、元素 |
| 自然界的水 | 水的净化、电解水、氢气 |
| 化学方程式 | 质量守恒定律、方程书写与计算 |
| 碳和碳的氧化物 | 金刚石/石墨/C60、二氧化碳、一氧化碳 |
| 燃料及其利用 | 燃烧条件、灭火原理、能源 |
| 金属和金属材料 | 金属活动性、酸碱盐基础 |

## ⚠️ 学习要点

- **化学用语**：元素符号、化学式、化学方程式必须过关
- **实验操作**：规范操作、安全意识
- **计算基础**：化学方程式计算是中考必考
