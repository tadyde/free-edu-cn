# 初中数学

> 匹配**人教版教材**，从算术迈入代数与几何的思维进阶。

## 📖 三年知识体系

| 年级 | 代数 | 几何 | 其他 |
|------|------|------|------|
| 7年级 | 有理数·整式·一元一次方程 | 图形认识·相交线平行线 | 数据收集 |
| 8年级 | 整式乘除·因式分解·分式·二次根式 | 三角形·全等·轴对称·勾股 | 一次函数 |
| 9年级 | 一元二次方程·反比例函数·三角函数 | 相似·圆·投影与视图 | 概率·统计 |

## ⚠️ 关键转折点

- **8年级上学期**：数学难度大幅提升，两极分化开始
- **因式分解**：贯穿整个初中，是代数基础
- **几何证明**：模型思维、辅助线思路是关键
