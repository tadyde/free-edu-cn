# 初中道德与法治

## 📖 三年学习重点

| 年级 | 心理健康 | 道德与法治 |
|------|----------|----------|
| 7年级 | 认识自我/学会交往/青春成长 | 自尊自信/自立自强 |
| 8年级 | 社会规则/社会秩序/社会责任感 | 走进法治生活/公民权利义务 |
| 9年级 | 认识国情/承担责任/走向未来的少年 | 改革开放/一国两制/中国梦 |
