# 初中英语

> 匹配**人教版英语教材（Go for it!）**，7-9年级英语能力系统提升。

## 📖 三年学习重点

| 年级 | 语法重点 | 词汇要求 |
|------|----------|----------|
| 7年级 | 一般现在时、现在进行时、情态动词 | 约1200词 |
| 8年级 | 一般过去时、一般将来时、形容词/副词比较级 | 约1500词 |
| 9年级 | 现在完成时、被动语态、定语从句 | 约1800词（中考词汇） |

| 题型 | 占比 |
|------|------|
| 听力 | 约20% |

| 阅读理解 | 约30% |
