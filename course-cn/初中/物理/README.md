# 初中物理

> 匹配**人教版物理教材**，8-9年级物理入门与进阶。

## 📖 三年知识体系

| 年级 | 内容重点 |
|------|----------|
| 8年级上 | 机械运动、声现象、物态变化、光现象、透镜 |
| 8年级下 | 力与运动、压强、浮力、功和机械能 |
| 9年级 | 电流与电路、欧姆定律、电功率、电与磁、信息的传递 |

## 📋 中考物理分值分布（参考）

| 模块 | 占比 |
|------|------|
| 力学（运动/力/压强/浮力/功/机械） | 约40% |
| 电学（电流/电压/电阻/欧姆/电功/电功率） | 约35% |
| 热学/光学/声学 | 约25% |
