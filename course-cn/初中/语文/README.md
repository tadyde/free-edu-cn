# 初中语文

> 匹配**部编版教材**，7-9年级语文能力系统培养。

## 📖 三年学习重点

| 年级 | 阅读重点 | 写作重点 |
|------|----------|----------|
| 7年级 | 记叙文精读、整体感知 | 记事写人、描写技巧 |
| 8年级 | 说明文、议论文入门 | 托物言志、写景抒情 |
| 9年级 | 综合性学习、文言文深化 | 议论文、中考作文 |

## 📋 考试题型分布（参考）

- 基础知识（字音/字形/词语/病句）：约20%
- 古诗文阅读（课内+课外）：约25%
- 文言文阅读：约15%
- 现代文阅读（说明文/议论文）：约20%
- 写作：约20%
