# 小学英语

> 匹配**PEP/外研版教材**，3-6年级英语启蒙与基础能力培养。

## 📖 内容结构

| 分类 | 说明 |
|------|------|
| 知识点清单 | 字母→词汇→句型→语法体系梳理 |
| 核心词汇 | 各年级课标要求核心词汇表 |
| 基本句型 | 6大基本句型 + 变化规则 |
| 常用介词 | 方位介词/时间介词/方式介词 |

## 📋 各年级重点

| 年级 | 核心能力 |
|------|----------|
| 3年级 | 字母认读、基础词汇、日常问候语 |
| 4年级 | 简单句型、读写入门、故事阅读 |
| 5-6年级 | 复合句入门、短文写作、考试备考 |
