## 📖 内容结构

| 分类 | 说明 |
|------|------|
| 知识点清单 | 按年级章节梳理，字词句段篇全覆盖 |
| 阅读理解 | 各文体（记叙文/说明文/议论文）答题技巧 |

| 年级 | 核心能力 |
|------|----------|
| 1-2年级 | 识字写字、看图说话、拼音基础 |
| 3-4年级 | 阅读理解入门、写作框架、古诗词积累 |
| 5-6年级 | 文章分析、深度写作、文言文入门 |
