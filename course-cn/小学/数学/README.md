# 小学数学

> 匹配**人教版教材**，1-6年级计算能力与数学思维同步培养。

## 📖 内容结构

| 分类 | 说明 |
|------|------|
| 知识点清单 | 整数→小数→分数→百分数体系完整梳理 |
| 公式汇总 | 周长/面积/体积/单位换算速查表 |
| 计算训练 | 各年级口算、竖式、混合运算 |
| 应用题 | 和差倍问题/行程问题/工程问题等 |
| 思维训练 | 奥数入门/逻辑推理/规律探索 |

## 📋 各年级重点

| 年级 | 核心内容 |
|------|----------|
| 1-2年级 | 20以内加减法、乘法口诀、认识图形 |
| 3-4年级 | 多位数乘除法、面积计算、统计图表 |
| 5-6年级 | 分数运算、比例、圆柱圆锥、综合应用 |
