# 🇨🇳 国内标准教育主线

> 本目录收录与**国家课程标准**同步的知识点、练习题和学习资料，完全匹配国内教材体系。

## 📚 学段导航

| 学段 | 学科 | 内容说明 |
|------|------|----------|
| [小学](./小学/) | 语文 · 数学 · 英语 | 基础知识 · 计算训练 · 古诗词 · 阅读写作 |
| [初中](./初中/) | 语 · 数 · 英 · 物 · 化 · 生 · 史 · 地 · 政 | 知识点 · 专题练习 · 公式汇总 · 实验操作 |
| [高中](./高中/) | 语 · 数 · 英 · 物 · 化 · 生 · 史 · 地 · 政 | 核心考点 · 公式速查 · 专题突破 · 应试技巧 |

## 📖 内容分类

每个学科目录下包含：
- **知识点清单** — 对标教材章节，知识点全覆盖
- **专题练习** — 常考题型、难点突破
- **知识归纳** — 公式速记、概念对比、易错点整理
- **写作/词汇** — 专项积累与训练

## 🎯 使用说明

1. 按学段 → 学科 → 内容类型查找资料
2. 优先使用 **知识点清单** 梳理框架
3. 用 **专题练习** 巩固薄弱环节
4. 所有内容永久免费，无需注册
