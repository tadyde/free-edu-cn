## 说明

大学学科众多，本专区不按具体专业细分，而是提供：
1. 各专业通用的公共课学习资料
2. 编程和计算机基础（几乎所有专业都需要）
3. 学习方法与效率提升
4. 考研、考证等备考资源
5. 职业发展指导

如需特定专业的资料，欢迎贡献！
