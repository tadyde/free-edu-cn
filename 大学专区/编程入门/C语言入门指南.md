# C语言入门指南

## 一、基本结构

`c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
`

## 二、数据类型

| 类型 | 字节数 | 范围 |
|------|--------|------|
| char | 1 | -128~127 |
| int | 4 | -2^31~2^31-1 |
| float | 4 | 约6位精度 |
| double | 8 | 约15位精度 |

## 三、运算符

- 算术：+ - * / %
- 关系：== != < > <= >=
- 逻辑：&& || !
- 自增自减：++ --

## 四、控制结构

### 条件
`c
if (condition) {
    // ...
} else if (condition2) {
    // ...
} else {
    // ...
}
`

### 循环
`c
for (int i = 0; i < n; i++) { }
while (condition) { }
do { } while (condition);
`

### switch
`c
switch (x) {
    case 1: break;
    case 2: break;
    default: break;
}
`

## 五、数组与字符串

`c
int arr[10];
char str[] = "hello";
`

## 六、函数

`c
int add(int a, int b) {
    return a + b;
}
`

## 七、指针

`c
int x = 10;
int *p = &x;
printf("%d", *p);  // 输出10
`

## 八、结构体

`c
struct Student {
    char name[20];
    int age;
    float score;
};
`

## 九、文件操作

`c
FILE *fp = fopen("test.txt", "r");
fclose(fp);
`

## 十、编译运行

`ash
gcc -o program program.c
./program

# 考研备考指南

## 一、时间规划

### 基础阶段（3-6月）
- 确定目标院校和专业
- 英语：背单词+长难句
- 数学：课本+基础习题
- 专业课：通读教材

### 强化阶段（7-9月）
- 英语：真题阅读+翻译
- 数学：强化班+习题集
- 政治：开始基础班
- 专业课：重点章节深入

### 冲刺阶段（10-12月）
- 全科真题模拟
- 政治背诵大题
- 英语作文模板
- 专业课查缺补漏

## 二、各科建议

### 政治（满分100）
- 选择题是关键，占50分
- 马原理解为主，毛概史纲记忆为主
- 大题最后一个月背诵

### 英语（满分100）
- 阅读理解最重要
- 单词每天背，不能停
- 作文准备模板

### 数学（满分150）
- 题海战术有效但要总结
- 错题本是利器
- 计算能力必须练

### 专业课（满分150/300）
- 找到目标院校真题
- 关注导师研究方向
- 历年真题反复做

## 三、重要时间节点

| 时间 | 事项 |
|------|------|
| 9月 | 预报名 |
| 10月 | 正式报名 |
| 11月 | 现场确认 |
| 12月底 | 初试 |
| 次年3-4月 | 复试 |

## 四、心态调整

- 保持规律作息
- 适度运动减压
- 不要和别人比进度
- 相信坚持的力量
