# 大学概率论与数理统计知识点清单

## 一、概率基础

### 1. 随机事件
- 样本空间、样本点
- 事件的关系：包含、相等、互斥、对立
- 事件运算：并(和)、交(积)、差、补

### 2. 概率定义
- 古典概型：P(A) = m/n
- 几何概型：P(A) = 区域度量/总度量
- 条件概率：P(A|B) = P(AB)/P(B)
- 乘法公式：P(AB) = P(A)P(B|A)
- 全概率公式：P(B) = sum(P(Ai)P(B|Ai))
- 贝叶斯公式：P(Ai|B) = P(Ai)P(B|Ai)/sum(P(Aj)P(B|Aj))

### 3. 独立性
- P(AB) = P(A)P(B)
- 互斥不一定独立，独立不一定互斥

## 二、随机变量

### 离散型
- 0-1分布：P(X=k) = p^k(1-p)^(1-k)
- 二项分布：X~B(n,p)
- 泊松分布：X~P(lambda)
- 几何分布：首次成功所需次数

### 连续型
- 均匀分布：X~U(a,b)
- 指数分布：f(x) = lambda*e^(-lambda*x)
- 正态分布：X~N(mu,sigma^2)
  - 标准化：Z=(X-mu)/sigma ~ N(0,1)

## 三、数字特征

- 期望：E(X)
- 方差：D(X) = E(X^2) - [E(X)]^2
- 协方差：Cov(X,Y) = E(XY) - E(X)E(Y)
- 相关系数：rho = Cov(X,Y)/sqrt(D(X)D(Y))

## 四、大数定律与中心极限定理

- 切比雪夫不等式
- 大数定律：频率趋于概率
- 中心极限定理：大量独立同分布随机变量之和近似正态

## 五、数理统计

- 统计量：样本均值、样本方差
- 三大分布：卡方分布、t分布、F分布
- 参数估计：矩估计、最大似然估计
- 区间估计
- 假设检验
