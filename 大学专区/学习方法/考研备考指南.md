# 考研备考指南

## 时间规划

### 基础阶段（3-6月）
- 确定目标院校和专业
- 英语：背单词+长难句
- 数学：课本+基础习题
- 专业课：通读教材

### 强化阶段（7-9月）
- 英语：真题阅读+翻译
- 数学：强化班+习题集
- 政治：开始基础班
- 专业课：重点章节深入

### 冲刺阶段（10-12月）
- 全科真题模拟
- 政治背诵大题
- 英语作文模板
- 专业课查缺补漏

## 各科建议

### 政治（满分100）
- 选择题是关键，占50分
- 马原理解为主，毛概史纲记忆为主
- 大题最后一个月背诵

### 英语（满分100）
- 阅读理解最重要
- 单词每天背不能停
- 作文准备模板

### 数学（满分150）
- 题海战术有效但要总结
- 错题本是利器
- 计算能力必须练

### 专业课（满分150/300）
- 找到目标院校真题
- 关注导师研究方向
- 历年真题反复做

## 重要时间节点

| 时间 | 事项 |
|------|------|
| 9月 | 预报名 |
| 10月 | 正式报名 |
| 11月 | 现场确认 |
| 12月底 | 初试 |
| 次年3-4月 | 复试 |

## 心态调整

- 保持规律作息
- 适度运动减压
- 不要和别人比进度
- 相信坚持的力量
