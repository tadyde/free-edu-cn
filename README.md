## 项目简介

本仓库致力于为全国小学生和初中生提供完全免费、无广告的学习资源。所有资料均以开源方式共享，任何人都可以使用、分享和贡献。

## 资源目录

### 小学专区
| 学科 | 内容 |
|------|------|
| [语文](小学专区/语文/) | 课文解读、作文指导、古诗词积累 |
| [数学](小学专区/数学/) | 计算训练、应用题、思维拓展 |
| [英语](小学专区/英语/) | 单词记忆、语法入门、听力练习 |

### 初中专区
| 学科 | 内容 |
|------|------|
| [语文](初中专区/语文/) | 阅读理解、文言文、写作技巧 |
| [数学](初中专区/数学/) | 代数、几何、压轴题解析 |
| [英语](初中专区/英语/) | 语法体系、阅读训练、写作模板 |
| [物理](初中专区/物理/) | 力学、电学、实验专题 |
| [化学](初中专区/化学/) | 元素周期、化学方程、实验要点 |

## 使用方式

1. 直接浏览仓库中对应学科目录
2. 下载所需资料，离线使用
3. 无需注册、无需付费、无任何广告
