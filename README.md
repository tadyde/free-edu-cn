> 本项目为全国学生提供**完全免费、无广告**的学习资料，覆盖小学至大学阶段。分为**国内标准线**和**国际融合线**两条独立体系。

---

## 📂 双轨体系

### 🇨🇳 国内标准线 · course-cn
与**国家课程标准**同步，完全匹配国内教材体系。

| 学段 | 学科 | 内容说明 |
|------|------|----------|
| [小学](course-cn/小学/) | 语文·数学·英语 | 基础知识/计算训练/古诗词/阅读写作 |
| [初中](course-cn/初中/) | 语·数·英·物·化·生·史·地·政 | 知识点·专题练习·公式汇总·实验操作 |
| [高中](course-cn/高中/) | 语·数·英·物·化·生·史·地·政 | 核心考点·公式速查·专题突破·高考备考 |

### 🌍 国际融合线 · course-global
收录**海外优质公开教育资源**，中外思维对比与国际视野拓展。

| 学段 | 覆盖内容 |
|------|----------|
| [小学](course-global/小学/) | CCSS/NC体系/日本教育/优质平台 |
| [初中](course-global/初中/) | IB MYP/GCSE入门/AP衔接/国际课程 |
| [高中](course-global/高中/) | AP/IB DP/A-Level/BC课程/国际备考 |
| [优质资源汇总](course-global/优质资源汇总/) | 免费平台·YouTube频道·中外对比 |
