## 项目简介

本仓库是全国学生的**完全免费、无广告**学习资料库，覆盖小学至大学阶段。所有资料依据国家课程标准（2022版）和统编版/人教版教材整理。

- 🔗 **在线访问**：[https://tadyde.github.io/free-edu-cn/](https://tadyde.github.io/free-edu-cn/)
- 📱 **微信小程序**：（开发中，将支持离线查阅与下载）
- 📂 **仓库地址**：[https://github.com/tadyde/free-edu-cn](https://github.com/tadyde/free-edu-cn)
