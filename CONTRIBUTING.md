# 贡献指南

感谢你对免费公益教育项目「哈腰章鱼」的贡献！

## 许可证说明

本项目采用 **GNU GPLv3** 许可证（详见 [LICENSE](LICENSE) 文件）。

贡献本项目即表示你同意：
- 你的贡献将以 GPLv3 协议发布
- 所有衍生作品必须保持开源，禁止闭源收费
- 需保留「哈腰章鱼项目」版权声明

## 如何贡献

### 1. 提交学习资料

- Fork 本仓库
- 在对应学科目录下添加资料
- 提交 Pull Request

### 2. 资料格式要求

- 优先使用 Markdown（.md）格式
- 如含图片，请放在同目录的 images 文件夹中
- 文件命名清晰，例如：初一数学-一元一次方程-基础题.md
- 每份资料开头注明：适用年级、知识点、难度

### 3. 内容规范

- 必须为**原创内容**或**已获授权**的转载
- 确保内容**准确无误**
- 禁止包含任何广告、推广链接
- 禁止包含收费引导
- 禁止提交受版权保护的收费资料

### 4. 提交 Issue

- 发现错误？提交 Issue 指出
- 想要某方面资料？提交 Issue 说明需求
